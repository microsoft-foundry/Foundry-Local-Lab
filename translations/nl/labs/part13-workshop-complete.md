# Deel 13: Workshop Voltooid

> **Gefeliciteerd!** Je hebt de volledige Foundry Local Workshop voltooid.

Je bent gegaan van het installeren van de CLI naar het bouwen van chatapplicaties, RAG-pijplijnen, multi-agent systemen, spraak-naar-tekst transcriptie, het compileren van je eigen aangepaste modellen, het inschakelen van tool-calling, en het aansluiten van een live web UI voor je multi-agent applicatie. Dit alles draait volledig op je eigen apparaat.

## Verdere Ideeën

- Combineer tool calling met agents om autonome workflows te bouwen
- Query een lokale database of roep interne REST APIs aan vanuit je tools
- Probeer verschillende modellen (`phi-4-mini`, `deepseek-r1-7b`) om kwaliteit en snelheid te vergelijken
- Voeg authenticatie of rate limiting toe aan je Zava Writer webserver
- Maak je eigen multi-agent applicatie voor een domein dat jou interesseert
- Zet uit in de cloud door Foundry Local te vervangen door Azure AI Foundry, dezelfde code, ander eindpunt
