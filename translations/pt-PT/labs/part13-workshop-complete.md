# Parte 13: Oficina Concluída

> **Parabéns!** Concluiu a Oficina completa do Foundry Local.

Passou desde a instalação do CLI até à construção de aplicações de chat, pipelines RAG, sistemas multi-agente, transcrição de voz para texto, compilação dos seus próprios modelos personalizados, ativação de chamadas de ferramentas e ligação de uma interface web em tempo real para a sua aplicação multi-agente. Tudo a correr inteiramente no seu dispositivo.

## Ideias Adicionais

- Combine chamadas de ferramentas com agentes para construir workflows autónomos
- Interrogue uma base de dados local ou chame APIs REST internas a partir das suas ferramentas
- Experimente modelos diferentes (`phi-4-mini`, `deepseek-r1-7b`) para comparar qualidade e velocidade
- Adicione autenticação ou limitação de taxa ao seu servidor web do Zava Writer
- Crie a sua própria aplicação multi-agente para um domínio que lhe interesse
- Desdobre para a cloud trocando o Foundry Local pelo Azure AI Foundry, mesmo código, endpoint diferente
