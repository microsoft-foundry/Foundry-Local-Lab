## Ideias Futuros

- Combine chamadas de ferramentas com agentes para construir fluxos de trabalho autônomos
- Consulte um banco de dados local ou chame APIs REST internas a partir das suas ferramentas
- Experimente diferentes modelos (`phi-4-mini`, `deepseek-r1-7b`) para comparar qualidade e velocidade
- Adicione autenticação ou limitação de taxa no servidor web do Zava Writer
- Crie sua própria aplicação multiagente para um domínio que você se importe
- Faça deploy na nuvem trocando o Foundry Local pelo Azure AI Foundry, mesmo código, endpoint diferente
