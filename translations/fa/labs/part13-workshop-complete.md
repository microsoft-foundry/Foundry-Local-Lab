# بخش ۱۳: کارگاه کامل شد

> **تبریک!** شما کارگاه کامل Foundry Local را به پایان رساندید.

شما از نصب CLI شروع کرده‌اید تا ساخت برنامه‌های چت، خط لوله‌های RAG، سیستم‌های چندعاملی، تبدیل گفتار به متن، کامپایل مدل‌های سفارشی خود، فعال‌سازی فراخوانی ابزارها، و راه‌اندازی رابط کاربری زنده وب برای برنامه چندعاملی خود. همه این‌ها کاملاً روی دستگاه شما اجرا می‌شود.

---

## آنچه ساختید

| بخش | آنچه ساختید |
|------|---------------|
| ۱ | نصب Foundry Local، کاوش مدل‌ها از طریق CLI |
| ۲ | تسلط بر API کیت توسعه Foundry Local برای مدیریت سرویس، کاتالوگ، کش و مدل |
| ۳ | اتصال از Python، JavaScript و C# با استفاده از SDK با OpenAI |
| ۴ | ساخت خط لوله RAG با بازیابی دانش محلی |
| ۵ | ایجاد عامل‌های هوش مصنوعی با شخصیت و خروجی ساختاریافته |
| ۶ | هماهنگی خط لوله‌های چندعاملی با حلقه‌های بازخورد |
| ۷ | بررسی برنامه پایانی تولید، نویسنده خلاق زوا |
| ۸ | ساخت گردش‌های کاری توسعه مبتنی بر ارزیابی برای عامل‌ها |
| ۹ | رونویسی صدا با Whisper، تبدیل گفتار به متن روی دستگاه |
| ۱۰ | کامپایل و اجرای مدل سفارشی Hugging Face با ONNX Runtime GenAI |
| ۱۱ | فعال‌سازی مدل‌های محلی برای فراخوانی توابع خارجی با فراخوانی ابزار |
| ۱۲ | ساخت رابط کاربری مبتنی بر مرورگر برای نویسنده خلاق زوا با پخش زنده در زمان واقعی |

---

## ایده‌های بیشتر

- ترکیب فراخوانی ابزار با عامل‌ها برای ساخت گردش‌های کاری خودکار
- پرس‌وجو از پایگاه داده محلی یا فراخوانی APIهای داخلی REST از ابزارهای خود
- امتحان مدل‌های مختلف (`phi-4-mini`، `deepseek-r1-7b`) برای مقایسه کیفیت و سرعت
- افزودن احراز هویت یا محدودیت نرخ به سرور وب نویسنده زوا
- ساخت برنامه چندعاملی خود برای حوزه‌ای که به آن علاقه دارید
- استقرار در فضای ابری با جایگزینی Foundry Local با Azure AI Foundry، همان کد، نقطه پایان متفاوت
