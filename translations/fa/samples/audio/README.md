# نمونه فایل‌های صوتی برای بخش ۷ - تبدیل گفتار به متن ویس Whisper

این فایل‌های WAV با استفاده از `pyttsx3` (تبدیل متن به گفتار Windows SAPI5) و با موضوع **محصولات DIY زاوا** از نسخه نمایشی نویسنده خلاق تولید شده‌اند.

## تولید نمونه‌ها

```bash
# از ریشه مخزن - نیاز به دایرکتوری .venv با نصب pyttsx3 دارد
.venv\Scripts\Activate.ps1          # ویندوز
python samples/audio/generate_samples.py
```

## فایل‌های نمونه

| فایل | سناریو | مدت زمان |
|------|--------|----------|
| `zava-customer-inquiry.wav` | مشتری درباره **دریل بی‌سیم Zava ProGrip** - گشتاور، عمر باتری، کیف حمل پرسش می‌کند | ~۱۵ ثانیه |
| `zava-product-review.wav` | مشتری درباره **رنگ داخلی UltraSmooth زاوا** - پوشش، زمان خشک شدن، VOC کم بررسی می‌کند | ~۲۲ ثانیه |
| `zava-support-call.wav` | تماس پشتیبانی درباره **جعبه ابزار TitanLock زاوا** - کلیدهای جایگزین، روکش‌های اضافی کشو | ~۲۰ ثانیه |
| `zava-project-planning.wav` | برنامه‌ریزی یک عرشه حیاط عقب با **کامپوزیت EcoBoard زاوا** و چراغ‌های BrightBeam توسط DIYer | ~۲۵ ثانیه |
| `zava-workshop-setup.wav` | مرور یک کارگاه کامل با استفاده از **هر پنج محصول زاوا** | ~۳۲ ثانیه |
| `zava-full-project-walkthrough.wav` | مرور بازسازی گسترده گاراژ با استفاده از **تمام محصولات زاوا** (برای تست صوتی طولانی، ببینید [Foundry-Local#517](https://github.com/microsoft/Foundry-Local/issues/517)) | ~۴ دقیقه |

## نکات

- فایل‌های WAV به مخزن **کامیت** شده‌اند (در `. فهرست شده‌اند). برای ساخت فایل‌های .wav جدید، اسکریپت بالا را اجرا کنید تا اسکریپت‌های جدید بازتولید شوند یا برای ساخت اسکریپت‌های جدید ویرایش شود.
- این اسکریپت از صدای **Microsoft David** (انگلیسی آمریکا) با سرعت ۱۶۰ کلمه در دقیقه برای نتایج واضح تبدیل گفتار به متن استفاده می‌کند.
- همه سناریوها به محصولات موجود در [`zava-creative-writer-local/src/api/agents/writer/products.json`](../../../../zava-creative-writer-local/src/api/agents/writer/products.json) ارجاع می‌دهند.
