## Ideas Adicionales

- Combina llamadas a herramientas con agentes para construir flujos de trabajo autónomos
- Consulta una base de datos local o llama APIs REST internas desde tus herramientas
- Prueba diferentes modelos (`phi-4-mini`, `deepseek-r1-7b`) para comparar calidad y velocidad
- Añade autenticación o limitación de tasa a tu servidor web Zava Writer
- Crea tu propia aplicación multiagente para un dominio que te interese
- Despliega en la nube cambiando Foundry Local por Azure AI Foundry, mismo código, diferente endpoint
