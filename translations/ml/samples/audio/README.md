ഈ WAV ഫയലുകൾ `pyttsx3` (Windows SAPI5 ടെക്സ്റ്റ്-ടു-സ്പീച്ച്) ഉപയോഗിച്ച് സൃഷ്ടിക്കപ്പെട്ടതാണ്, ക്രിയേറ്റീവ് റൈറ്റർ ഡെമോയിൽ നിന്നുള്ള **Zava DIY ഉൽപ്പന്നങ്ങൾ** ചുറ്റിപ്പറ്റിയാണ് വിഷയം.

## സാമ്പിളുകൾ നിർമിക്കുക

```bash
# റിപൊ നീ്ഊലം - pyttsx3 ഇൻസ്റ്റാൾ ചെയ്ത .venv ആവശ്യമാണ്
.venv\Scripts\Activate.ps1          # വിൻഡോസ്
python samples/audio/generate_samples.py
```
