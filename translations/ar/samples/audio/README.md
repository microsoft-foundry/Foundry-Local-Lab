# ملفات صوتية نموذجية للجزء 7 - نسخ الصوت Whisper

يتم إنشاء هذه الملفات بصيغة WAV باستخدام `pyttsx3` (تحويل النص إلى كلام بنظام Windows SAPI5) وموضوعها حول **منتجات زافا للأعمال اليدوية** من عرض الكاتب المبدع.

## إنشاء العينات

```bash
# من جذر المستودع - يتطلب .venv مع تثبيت pyttsx3
.venv\Scripts\Activate.ps1          # ويندوز
python samples/audio/generate_samples.py
```

## ملفات نموذجية

| الملف | السيناريو | المدة |
|------|----------|----------|
| `zava-customer-inquiry.wav` | زبون يسأل عن **مثقاب زافا بروغريب اللاسلكي** - عزم الدوران، عمر البطارية، حقيبة الحمل | ~15 ثانية |
| `zava-product-review.wav` | مراجعة زبون لـ **طلاء داخلي زافا ألتراسموث** - التغطية، وقت الجفاف، انبعاثات المركبات العضوية المتطايرة المنخفضة | ~22 ثانية |
| `zava-support-call.wav` | مكالمة دعم حول **صندوق أدوات زافا تايتانلوك** - مفاتيح بديلة، بطانات أدراج إضافية | ~20 ثانية |
| `zava-project-planning.wav` | هاوٍ يخطط لبناء منصة في الحديقة باستخدام **زافا إيكوبورد لتكسية الأرصفة المركبة** وأضواء برايت بيم | ~25 ثانية |
| `zava-workshop-setup.wav` | جولة تعريفية على ورشة كاملة تستخدم **جميع منتجات زافا الخمسة** | ~32 ثانية |
| `zava-full-project-walkthrough.wav` | جولة تفصيلية لترميم المرآب باستخدام **جميع منتجات زافا** (لاختبار ملفات صوتية طويلة، راجع [Foundry-Local#517](https://github.com/microsoft/Foundry-Local/issues/517)) | ~4 دقائق |

## ملاحظات

- ملفات WAV تم **تضمينها** في المستودع (مذكورة في . لإنشاء ملفات .wav جديدة، نفّذ السكريبت أعلاه لإعادة توليد نصوص جديدة أو تعديلها لإنشاء نصوص جديدة.
- يستخدم السكريبت صوت **مايكروسوفت ديفيد** (الإنجليزية الأمريكية) بسرعة 160 كلمة في الدقيقة لنتائج نسخ واضحة.
- جميع السيناريوهات تشير إلى منتجات من [`zava-creative-writer-local/src/api/agents/writer/products.json`](../../../../zava-creative-writer-local/src/api/agents/writer/products.json).
