# Dio 13: Radionica završena

> **Čestitamo!** Završili ste cijelu Foundry Local radionicu.

Prošli ste od instalacije CLI-ja do izgradnje chat aplikacija, RAG pipelinea, sustava s više agenata, pretvaranja govora u tekst, kompajliranja vlastitih prilagođenih modela, omogućavanja poziva alata i povezivanja živog web sučelja za vašu aplikaciju s više agenata. Sve to radi u potpunosti na vašem uređaju.

---

## Što ste izgradili

| Dio | Što ste izgradili |
|------|---------------|
| 1 | Instalirali Foundry Local, istražili modele putem CLI-ja |
| 2 | Savladali Foundry Local SDK API za upravljanje servisima, katalogom, kešom i modelima |
| 3 | Povezali se iz Pythona, JavaScripta i C# koristeći SDK s OpenAI |
| 4 | Izgradili RAG pipeline s lokalnim dohvatom znanja |
| 5 | Stvorili AI agente s personama i strukturiranim izlazima |
| 6 | Orkestrirali pipeline s više agenata s povratnim informacijama |
| 7 | Istražili produkcijsku završnu aplikaciju, Zava Creative Writer |
| 8 | Izgradili tijekove razvoja vođene evaluacijom za agente |
| 9 | Transkribirali audio s Whisperom, govor u tekst na uređaju |
| 10 | Kompajlirali i pokrenuli prilagođeni Hugging Face model s ONNX Runtime GenAI |
| 11 | Omogućili lokalnim modelima pozivanje vanjskih funkcija putem poziva alata |
| 12 | Izgradili web sučelje za Zava Creative Writer s prijenosom uživo u stvarnom vremenu |

---

## Daljnje ideje

- Kombinirajte pozivanje alata s agentima za izgradnju autonomnih tijekova rada
- Upitujte lokalnu bazu podataka ili pozivajte interne REST API-je iz svojih alata
- Isprobajte različite modele (`phi-4-mini`, `deepseek-r1-7b`) za usporedbu kvalitete i brzine
- Dodajte autentifikaciju ili ograničenje brzine na svoj web poslužitelj Zava Writer
- Kreirajte vlastitu aplikaciju s više agenata za područje koje vas zanima
- Postavite u oblak zamjenom Foundry Local s Azure AI Foundry, isti kod, druga krajnja točka
