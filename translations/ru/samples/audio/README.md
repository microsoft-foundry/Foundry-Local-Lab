# Пример аудиофайлов для Части 7 - Распознавание голоса Whisper

Эти WAV-файлы созданы с использованием `pyttsx3` (текст-в-речь Windows SAPI5) и тематически связаны с **продуктами Zava DIY** из демонстрации Creative Writer.

## Генерация образцов

```bash
# Из корня репозитория - требуется .venv с установленным pyttsx3
.venv\Scripts\Activate.ps1          # Windows
python samples/audio/generate_samples.py
```

## Пример файлов

| Файл | Сценарий | Продолжительность |
|------|----------|-------------------|
| `zava-customer-inquiry.wav` | Клиент спрашивает о **беспроводной дрели Zava ProGrip** - крутящий момент, время работы аккумулятора, переносной кейс | ~15 сек |
| `zava-product-review.wav` | Отзыв клиента о **краске для интерьера Zava UltraSmooth** - покрытие, время высыхания, низкое содержание ЛОС | ~22 сек |
| `zava-support-call.wav` | Звонок в поддержку по поводу **ящика для инструментов Zava TitanLock** - запасные ключи, дополнительные вкладыши для ящиков | ~20 сек |
| `zava-project-planning.wav` | Планирование проекта строительства заднего двора с использованием **композитной доски Zava EcoBoard** и освещения BrightBeam | ~25 сек |
| `zava-workshop-setup.wav` | Обзор полностью оснащенной мастерской с использованием **всех пяти продуктов Zava** | ~32 сек |
| `zava-full-project-walkthrough.wav` | Детальный обзор ремонта гаража с использованием **всех продуктов Zava** (для тестирования длинных аудио см. [Foundry-Local#517](https://github.com/microsoft/Foundry-Local/issues/517)) | ~4 мин |

## Заметки

- WAV-файлы **зафиксированы** в репозитории (список в `. Чтобы создать новые файлы .wav, запустите скрипт выше для генерации новых скриптов или измените его для создания новых скриптов.
- В скрипте используется голос **Microsoft David** (американский английский) с темпом 160 слов в минуту для четкой расшифровки.
- Все сценарии относятся к продуктам из [`zava-creative-writer-local/src/api/agents/writer/products.json`](../../../../zava-creative-writer-local/src/api/agents/writer/products.json).
