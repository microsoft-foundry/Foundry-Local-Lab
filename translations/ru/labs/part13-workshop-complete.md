## Дополнительные идеи

- Совмещайте вызов инструментов с агентами для создания автономных рабочих процессов
- Запрашивайте локальную базу данных или вызывайте внутренние REST API из ваших инструментов
- Пробуйте разные модели (`phi-4-mini`, `deepseek-r1-7b`) для сравнения качества и скорости
- Добавьте аутентификацию или ограничение скорости для вашего веб-сервера Zava Writer
- Создайте своё многоагентное приложение для интересующей вас сферы
- Развертывайте в облаке, заменив Foundry Local на Azure AI Foundry, тот же код, другой endpoint
