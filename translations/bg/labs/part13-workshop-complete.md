# Част 13: Работилницата е завършена

> **Поздравления!** Вие завършихте цялата работилница Foundry Local.

Вие преминахте от инсталиране на CLI до изграждане на чат приложения, RAG конвейери, мултиагентни системи, транскрибиране от реч в текст, компилиране на собствени модели, активиране на извикване на инструменти и свързване на жив уеб интерфейс за вашето мултиагентно приложение. Всичко това се изпълнява изцяло на вашето устройство.

---

## Какво построихте

| Част | Какво построихте |
|------|------------------|
| 1 | Инсталирахте Foundry Local, изследвахте модели чрез CLI |
| 2 | Усвоихте Foundry Local SDK API за управление на услуги, каталог, кеш и модели |
| 3 | Свързахте се от Python, JavaScript и C# с SDK и OpenAI |
| 4 | Построихте RAG конвейер с локално извличане на знания |
| 5 | Създадохте AI агенти с персонажи и структурирани изходи |
| 6 | Организирахте мултиагентни конвейери с обратни връзки |
| 7 | Изследвахте приложение за производствен край – Zava Creative Writer |
| 8 | Изградихте работни процеси за развитие, водени от оценяване на агентите |
| 9 | Транскрибирахте аудио с Whisper, разпознаване на реч на устройството |
| 10 | Компилирахте и стартирахте персонализиран Hugging Face модел с ONNX Runtime GenAI |
| 11 | Активирахте локални модели да извикват външни функции чрез tool calling |
| 12 | Построихте браузърен потребителски интерфейс за Zava Creative Writer с реалновременен стрийминг |

---

## Допълнителни идеи

- Комбинирайте tool calling с агенти за изграждане на автономни работни процеси
- Запитвайте локална база данни или извиквайте вътрешни REST API-та от вашите инструменти
- Пробвайте различни модели (`phi-4-mini`, `deepseek-r1-7b`), за да сравните качество и скорост
- Добавете автентикация или ограничение на честотата на заявки към уеб сървъра на Zava Writer
- Създайте свое мултиагентно приложение за област, която ви интересува
- Разположете в облака, като замените Foundry Local с Azure AI Foundry – същият код, различен крайна точка
