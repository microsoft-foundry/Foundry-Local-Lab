# Примерни аудио файлове за Част 7 - Транскрипция на глас с Whisper

Тези WAV файлове са генерирани с `pyttsx3` (Windows SAPI5 синтезатор на реч) и са тематично свързани с **продукти Zava DIY** от демонстрацията Creative Writer.

## Генериране на примерите

```bash
# От корена на хранилището - изисква .venv с инсталиран pyttsx3
.venv\Scripts\Activate.ps1          # Windows
python samples/audio/generate_samples.py
```

## Примерни файлове

| Файл | Сценарий | Продължителност |
|------|----------|-----------------|
| `zava-customer-inquiry.wav` | Клиент пита за **Zava ProGrip Кабелен винтоверт** - въртящ момент, живот на батерията, калъф за носене | ~15 сек |
| `zava-product-review.wav` | Клиент преглежда **Zava UltraSmooth Интериорна боя** - покритие, време за съхнене, ниски ЛОС | ~22 сек |
| `zava-support-call.wav` | Обаждане в поддръжка за **Zava TitanLock Куфар за инструменти** - резервни ключове, допълнителни постелки за чекмеджета | ~20 сек |
| `zava-project-planning.wav` | Самодеец планира бекярд палуба с **Zava EcoBoard Композитна настилка** и осветление BrightBeam | ~25 сек |
| `zava-workshop-setup.wav` | Разглеждане на цял работен цех с използване на **всички пет продукта Zava** | ~32 сек |
| `zava-full-project-walkthrough.wav` | Разширено разглеждане на ремонт на гараж с използване на **всички продукти Zava** (за тестване на дълги аудио файлове вижте [Foundry-Local#517](https://github.com/microsoft/Foundry-Local/issues/517)) | ~4 мин |

## Забележки

- WAV файловете са **включени** в хранилището (посочени в `. За създаване на нови .wav файлове, стартирайте скрипта по-горе за генериране на нови скриптове или ги модифицирайте, за да създадете нови скриптове.
- Скриптът използва глас на **Microsoft David** (американски английски) при 160 думи в минута за ясни резултати от транскрипцията.
- Всички сценарии се отнасят за продукти от [`zava-creative-writer-local/src/api/agents/writer/products.json`](../../../../zava-creative-writer-local/src/api/agents/writer/products.json).
