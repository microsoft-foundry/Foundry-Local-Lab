# 13. rész: Műhelymunka befejezve

> **Gratulálunk!** Teljesen befejezted a Foundry Local Műhelymunkát.

Az CLI telepítésétől eljutottál a csevegőalkalmazások, RAG-pipeline-ok, többszörös ügynök rendszerek, beszéd-szöveg átirat, egyedi modellek fordítása, eszközhívás engedélyezése és egy élő webes UI összekapcsolásáig a többszörös ügynökös alkalmazásodhoz. Mindez teljes egészében a saját eszközödön fut.

---

## Amit építettél

| Rész | Amit építettél |
|------|---------------|
| 1 | Telepítetted a Foundry Local-t, és felfedezted a modelleket a CLI-n keresztül |
| 2 | Mesteri szinten elsajátítottad a Foundry Local SDK API-t szolgáltatás, katalógus, cache és modellkezeléshez |
| 3 | Csatlakoztál Pythonból, JavaScriptből és C#-ból az SDK segítségével az OpenAI-hoz |
| 4 | Építettél egy RAG pipeline-t helyi tudáskinyeréssel |
| 5 | Létrehoztál AI ügynököket személyiségekkel és strukturált kimenettel |
| 6 | Orkhesztráltál többszörös ügynök pipeline-okat visszacsatolási hurkokkal |
| 7 | Felfedezted egy éles projekt csúcspontját, a Zava Kreatív Írót |
| 8 | Értékelés-vezérelt fejlesztési munkafolyamatokat hoztál létre az ügynökökhöz |
| 9 | Átírtad hanganyagot Whisper-rel, helyi beszédfelismeréssel |
| 10 | Lefordítottad és futtattad egyedi Hugging Face modellt ONNX Runtime GenAI segítségével |
| 11 | Engedélyezted, hogy a helyi modellek külső függvényeket hívjanak eszközhívással |
| 12 | Építettél egy böngésző-alapú UI-t a Zava Kreatív Íróhoz valós idejű adatfolyam kezeléssel |

---

## További ötletek

- Kombináld az eszközhívást ügynökökkel autonóm munkafolyamatok létrehozásához
- Kérdezz le helyi adatbázist, vagy hívd meg belső REST API-kat az eszközeidből
- Próbálj ki különböző modelleket (`phi-4-mini`, `deepseek-r1-7b`) a minőség és sebesség összehasonlításához
- Adj hozzá hitelesítést vagy sebességkorlátozást a Zava Író webkiszolgálódhoz
- Hozz létre egyéni többszörös ügynök alkalmazást egy számodra fontos területen
- Telepíts a felhőbe, lecserélve a Foundry Local-t Azure AI Foundry-re, ugyanazzal a kóddal, más végponttal
