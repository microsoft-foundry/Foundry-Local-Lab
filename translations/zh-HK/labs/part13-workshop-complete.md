# 第 13 部分：工作坊完成

> **恭喜！** 你已完成完整的 Foundry Local 工作坊。

## 進階想法

- 將工具調用與代理結合，以構建自主工作流程
- 從你的工具查詢本地數據庫或呼叫內部 REST API
- 嘗試不同模型（`phi-4-mini`、`deepseek-r1-7b`）比較質量與速度
- 為你的 Zava Writer 網頁伺服器新增身份驗證或速率限制
- 為你關心的領域創建自己的多代理應用程式
- 通過將 Foundry Local 換成 Azure AI Foundry 部署到雲端，相同的程式碼，不同的端點
