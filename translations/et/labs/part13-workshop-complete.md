# Osa 13: Töötuba lõpetatud

> **Palju õnne!** Sa oled läbinud kogu Foundry Local töötoa.

Sa alustasid CLI installimisest ning jõudsid vestlusrakenduste, RAG torujuhtmete, mitmeagendiliste süsteemide, kõnest tekstiks transkriptsiooni, oma kohandatud mudelite koostamise, tööriistakõnede võimaldamise ning oma mitmeagendilise rakenduse jaoks reaalajas veebikasutajaliidese loomise juurde. Kõik see töötab täielikult sinu seadmes.

---

## Mida sa ehitasid

| Osa | Mida sa ehitasid |
|------|------------------|
| 1 | Paigaldasid Foundry Local, uurisid mudeleid CLI kaudu |
| 2 | Valdasid Foundry Local SDK API teenuste, kataloogi, vahemäluga ja mudelite halduseks |
| 3 | Ühendusid Pythonist, JavaScriptist ja C# SDK kaudu OpenAI-ga |
| 4 | Ehitasid RAG torujuhtme kohalikus teadmiste päringus |
| 5 | Lõid tehisintellekti agente isikupärade ja struktureeritud väljundiga |
| 6 | Orkestreerisid mitmeagendilisi torujuhtmeid tagasiside silmustega |
| 7 | Uurisin tootmisvalmid lõpprakendust, Zava Creative Writerit |
| 8 | Lõid agentide hindamisega juhitud arendusvooge |
| 9 | Transkribeerisid heli Whisperiga, kõnest tekstiks seadmes endas |
| 10 | Kokkupandud ja jooksutasid kohandatud Hugging Face mudelit ONNX Runtime GenAI-ga |
| 11 | Võimaldasid kohalikel mudelitel välisfunktsioonide kutsumise tööriistadega |
| 12 | Ehitasid brauseripõhise kasutajaliidese Zava Creative Writerile reaalajas voogedastusega |

---

## Edasised ideed

- Ühenda tööriistakõned agentidega autonoomsete töövoogude loomiseks
- Küsi oma tööriistade kaudu kohalikust andmebaasist või kutsu sisemisi REST API-sid
- Proovi erinevaid mudeleid (`phi-4-mini`, `deepseek-r1-7b`), et võrrelda kvaliteeti ja kiirust
- Lisa oma Zava Writeri veebiserverisse autentimine või kasutuspiirangud
- Loo oma mitmeagendiline rakendus valdkonnas, mis sind huvitab
- Loo pilvepaigutus, vahetades Foundry Locali Azure AI Foundry vastu, kasutades sama koodi, kuid teist lõpp-punkti
