# Bahagian 13: Bengkel Lengkap

> **Tahniah!** Anda telah menyelesaikan Bengkel Foundry Local sepenuhnya.

Anda telah melalui pemasangan CLI hingga membina aplikasi chat, paip RAG, sistem multi-ejen, transkripsi pertuturan-ke-teks, pengkompilan model tersuai anda sendiri, mengaktifkan pemanggilan alat, dan menyerasikan UI web langsung untuk aplikasi multi-ejen anda. Semua ini berjalan sepenuhnya pada peranti anda.

## Idea Lanjut

- Gabungkan pemanggilan alat dengan ejen untuk membina aliran kerja automatik
- Tanya pangkalan data tempatan atau panggil API REST dalaman dari alat anda
- Cuba model berlainan (`phi-4-mini`, `deepseek-r1-7b`) untuk membandingkan kualiti dan kelajuan
- Tambah pengesahan atau had kadar pada pelayan web Zava Writer anda
- Cipta aplikasi multi-ejen anda sendiri untuk domain yang anda minati
- Sebarkan ke awan dengan menukar Foundry Local kepada Azure AI Foundry, kod sama, titik akhir berbeza
