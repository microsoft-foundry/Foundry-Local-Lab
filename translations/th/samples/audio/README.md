# ไฟล์เสียงตัวอย่างสำหรับ Part 7 - การถอดเสียงเสียง Whisper

ไฟล์ WAV เหล่านี้ถูกสร้างขึ้นโดยใช้ `pyttsx3` (Windows SAPI5 แปลงข้อความเป็นเสียง) และมีธีมเกี่ยวกับ **ผลิตภัณฑ์ Zava DIY** จากเดโม Creative Writer

## การสร้างตัวอย่าง

```bash
# จากโฟลเดอร์หลักของรีโพ - ต้องมี .venv ที่ติดตั้ง pyttsx3 แล้ว
.venv\Scripts\Activate.ps1          # วินโดวส์
python samples/audio/generate_samples.py
```

## ไฟล์ตัวอย่าง

| ไฟล์ | สถานการณ์ | ระยะเวลา |
|------|----------|----------|
| `zava-customer-inquiry.wav` | ลูกค้าสอบถามเกี่ยวกับ **สว่านไร้สาย Zava ProGrip** - แรงบิด, อายุแบตเตอรี่, กระเป๋าถือ | ~15 วินาที |
| `zava-product-review.wav` | ลูกค้ารีวิว **สีน้ำภายใน Zava UltraSmooth** - การปกคลุม, เวลาการแห้ง, ปริมาณสาร VOC ต่ำ | ~22 วินาที |
| `zava-support-call.wav` | การโทรติดต่อฝ่ายสนับสนุนเกี่ยวกับ **ตู้เครื่องมือ Zava TitanLock** - กุญแจสำรอง, แผ่นรองลิ้นชักเสริม | ~20 วินาที |
| `zava-project-planning.wav` | คนทำ DIY วางแผนทำระเบียงหลังบ้านด้วย **ไม้ระเบียงคอมโพสิต Zava EcoBoard** และไฟ BrightBeam | ~25 วินาที |
| `zava-workshop-setup.wav` | การเดินชมเวิร์กช็อปครบชุดที่ใช้ **ผลิตภัณฑ์ Zava ทั้ง 5 ชิ้น** | ~32 วินาที |
| `zava-full-project-walkthrough.wav` | การเดินชมการปรับปรุงโรงรถเต็มรูปแบบโดยใช้ **ผลิตภัณฑ์ Zava ทั้งหมด** (สำหรับทดสอบเสียงยาว ดูที่ [Foundry-Local#517](https://github.com/microsoft/Foundry-Local/issues/517)) | ~4 นาที |

## หมายเหตุ

- ไฟล์ WAV ถูก **คอมมิต** ลงในรีโป (แสดงใน `.`) เพื่อสร้างไฟล์ .wav ใหม่ให้รันสคริปต์ด้านบนเพื่อสร้างสคริปต์ใหม่หรือแก้ไขเพื่อสร้างสคริปต์ใหม่
- สคริปต์ใช้เสียง **Microsoft David** (ภาษาอังกฤษแบบสหรัฐฯ) ที่ความเร็ว 160 คำต่อนาที เพื่อผลลัพธ์การถอดเสียงที่ชัดเจน
- สถานการณ์ทั้งหมดอ้างอิงจากผลิตภัณฑ์ใน [`zava-creative-writer-local/src/api/agents/writer/products.json`](../../../../zava-creative-writer-local/src/api/agents/writer/products.json)
