# Μέρος 13: Ολοκλήρωση Εργαστηρίου

> **Συγχαρητήρια!** Έχετε ολοκληρώσει ολόκληρο το Εργαστήριο Foundry Local.

Έχετε προχωρήσει από την εγκατάσταση του CLI μέχρι την κατασκευή εφαρμογών συνομιλίας, pipelines RAG, συστημάτων πολλαπλών πρακτόρων, μεταγραφή ομιλίας σε κείμενο, τη σύνταξη δικών σας προσαρμοσμένων μοντέλων, την ενεργοποίηση κλήσης εργαλείων και τη σύνδεση ζωντανής διεπαφής ιστού για την εφαρμογή πολλαπλών πρακτόρων σας. Όλα αυτά να τρέχουν εξ ολοκλήρου στη συσκευή σας.

---

## Τι Δημιουργήσατε

| Μέρος | Τι Δημιουργήσατε |
|------|------------------|
| 1 | Εγκαταστήσατε το Foundry Local, εξερευνήσατε μοντέλα μέσω CLI |
| 2 | Κατακτήσατε το API του Foundry Local SDK για διαχείριση υπηρεσίών, καταλόγου, cache και μοντέλων |
| 3 | Συνδεθήκατε από Python, JavaScript, και C# χρησιμοποιώντας το SDK με OpenAI |
| 4 | Δημιουργήσατε pipeline RAG με τοπική ανάκτηση γνώσης |
| 5 | Δημιουργήσατε AI πράκτορες με προσωπικότητες και δομημένη έξοδο |
| 6 | Οργανώσατε pipelines πολλαπλών πρακτόρων με βρόχους ανατροφοδότησης |
| 7 | Εξερευνήσατε μια εφαρμογή ορόσημο παραγωγής, τον Zava Creative Writer |
| 8 | Δημιουργήσατε ροές εργασίας ανάπτυξης με αξιολογήσεις για τους πράκτορες |
| 9 | Μεταγράψατε ήχο με Whisper, ομιλία σε κείμενο στη συσκευή |
| 10 | Συντάξατε και εκτελέσατε ένα προσαρμοσμένο μοντέλο Hugging Face με ONNX Runtime GenAI |
| 11 | Ενεργοποιήσατε μοντέλα τοπικά να καλούν εξωτερικές λειτουργίες με κλήση εργαλείων |
| 12 | Δημιουργήσατε διεπαφή χρήστη σε πρόγραμμα περιήγησης για τον Zava Creative Writer με ζωντανή ροή δεδομένων |

---

## Περαιτέρω Ιδέες

- Συνδυάστε την κλήση εργαλείων με πράκτορες για να δημιουργήσετε αυτόνομες ροές εργασίας
- Ερωτήστε μια τοπική βάση δεδομένων ή καλέστε εσωτερικά REST APIs από τα εργαλεία σας
- Δοκιμάστε διαφορετικά μοντέλα (`phi-4-mini`, `deepseek-r1-7b`) για σύγκριση ποιότητας και ταχύτητας
- Προσθέστε πιστοποίηση ή περιορισμό ρυθμού στον web server του Zava Writer σας
- Δημιουργήστε τη δική σας εφαρμογή πολλαπλών πρακτόρων για ένα τομέα που σας ενδιαφέρει
- Αναπτύξτε στο cloud αντικαθιστώντας το Foundry Local με Azure AI Foundry, ίδιος κώδικας, διαφορετικό endpoint
