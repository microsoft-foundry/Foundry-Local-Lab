# Del 13: Workshop fuldført

> **Tillykke!** Du har gennemført hele Foundry Local Workshoppen.

Du er gået fra at installere CLI'en til at bygge chat-applikationer, RAG-pipelines, multi-agent systemer, tale-til-tekst transskription, kompilering af dine egne brugerdefinerede modeller, aktivering af tool calling og opsætning af et live web UI til din multi-agent applikation. Alt sammen kørende udelukkende på din enhed.

---

## Hvad du har bygget

| Del | Hvad du har bygget |
|------|---------------|
| 1 | Installerede Foundry Local, udforskede modeller via CLI |
| 2 | Mestrede Foundry Local SDK API til service, katalog, cache og modelstyring |

| 4 | Byggede en RAG-pipeline med lokal videnssøgning |

| 12 | Byggede et browserbaseret UI til Zava Creative Writer med realtids streaming |

---

## Yderligere idéer

- Kombinér tool calling med agenter for at bygge autonome workflows
- Forespørg en lokal database eller kald interne REST API'er fra dine værktøjer
- Prøv forskellige modeller (`phi-4-mini`, `deepseek-r1-7b`) for at sammenligne kvalitet og hastighed
- Tilføj autentifikation eller rate limiting til din Zava Writer webserver
- Skab din egen multi-agent applikation for et område, du interesserer dig for
- Deploy til skyen ved at skifte Foundry Local ud med Azure AI Foundry, samme kode, anden endpoint
