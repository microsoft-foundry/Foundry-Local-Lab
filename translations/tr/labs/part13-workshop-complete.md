# Bölüm 13: Atölye Tamamlandı

> **Tebrikler!** Foundry Local Atölyesini tamamen tamamladınız.

CLI kurulumu yapmaktan sohbet uygulamaları, RAG boru hatları, çoklu ajan sistemleri, konuşmadan metne transkripsiyon, kendi özel modellerinizi derlemeye, araç çağırmayı etkinleştirmeye ve çoklu ajan uygulamanız için canlı bir web UI bağlamaya kadar ilerlediniz. Hepsi tamamen cihazınızda çalışıyor.

---

## Neleri Yaptınız

| Bölüm | Yaptıklarınız |
|------|---------------|
| 1 | Foundry Local'ı kurdunuz, modelleri CLI üzerinden keşfettiniz |
| 2 | Servis, katalog, önbellek ve model yönetimi için Foundry Local SDK API'sini öğrendiniz |
| 3 | Python, JavaScript ve C# ile SDK kullanarak OpenAI'a bağlandınız |
| 4 | Yerel bilgi alma özellikli bir RAG boru hattı kurdunuz |
| 5 | Kişilik ve yapılandırılmış çıktı ile AI ajanları oluşturdunuz |
| 6 | Geri bildirim döngüleriyle çoklu ajan boru hatları düzenlediniz |
| 7 | Üretim aşaması uygulaması Zava Creative Writer’ı incelediniz |
| 8 | Ajanlar için değerlendirme odaklı geliştirme iş akışları kurdunuz |
| 9 | Whisper ile cihazda konuşmayı metne çevirdiniz |
| 10 | ONNX Runtime GenAI ile özel Hugging Face modelini derleyip çalıştırdınız |
| 11 | Araç çağırma ile yerel modellerin dış işlevleri kullanmasını sağladınız |
| 12 | Zava Creative Writer için gerçek zamanlı yayın özellikli tarayıcı tabanlı UI oluşturdunuz |

---

## Daha Fazla Fikir

- Araç çağırmayı ajanlarla birleştirerek otonom iş akışları oluşturun
- Araçlarınızdan yerel bir veritabanına sorgu gönderin veya dahili REST API’leri çağırın
- Kalite ve hız karşılaştırması için farklı modelleri deneyin (`phi-4-mini`, `deepseek-r1-7b`)
- Zava Writer web sunucunuza kimlik doğrulama veya hız sınırlaması ekleyin
- İlginizi çeken bir alan için kendi çoklu ajan uygulamanızı oluşturun
- Aynı kodla farklı uç nokta kullanarak Foundry Local yerine Azure AI Foundry’ye bulut dağıtımı yapın
