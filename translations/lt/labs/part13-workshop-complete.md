# 13 dalis: Dirbtuvės baigtos

> **Sveikiname!** Jūs baigėte visą Foundry Local dirbtuvių kursą.

Nuo CLI diegimo iki pokalbių programų kūrimo, RAG sistemų, daugiagentinių sistemų, balso į tekstą transkripcijos, savo pasirinktinių modelių kompiliavimo, įrankių iškvietimo įgalinimo ir tiesioginės internetinės UI prijungimo prie daugiagentės programos – visa tai veikia visiškai jūsų įrenginyje.

---

## Ką sukūrėte

| Dalis | Ką sukūrėte |
|------|---------------|
| 1 | Įdiegėte Foundry Local, tyrinėjote modelius per CLI |
| 2 | Įvaldėte Foundry Local SDK API paslaugų, katalogų, talpyklos ir modelių valdymui |
| 3 | Prisijungėte iš Python, JavaScript ir C# naudodami SDK su OpenAI |
| 4 | Sukūrėte RAG sistemą su vietiniu žinių gavimu |
| 5 | Sukūrėte AI agentus su personomis ir struktūruotu išvestimi |
| 6 | Koordinavote daugiagentines sistemas su atsiliepimų ciklais |
| 7 | Tyrinėjote gamybinį baigiamąjį projektą, Zava kūrybinio rašytojo programą |
| 8 | Sukūrėte vertinimu pagrįstą agentų kūrimo darbo eigą |
| 9 | Transkribavote garsą su Whisper, balso į tekstą tiesiogiai įrenginyje |
| 10 | Kompiliavote ir paleidote pasirinktą Hugging Face modelį su ONNX Runtime GenAI |
| 11 | Įgalinote vietinius modelius kviesti išorines funkcijas naudodami įrankių iškvietimą |
| 12 | Sukūrėte naršyklės pagrindu veikiančią UI Zava kūrybiniam rašytojui su realaus laiko transliacija |

---

## Tolimesnės idėjos

- Derinkite įrankių iškvietimą su agentais kuriant autonomines darbo eigas
- Klauskite vietinės duomenų bazės arba kvieskite vidines REST API iš savo įrankių
- Išbandykite skirtingus modelius (`phi-4-mini`, `deepseek-r1-7b`), kad palygintumėte kokybę ir greitį
- Pridėkite autentifikavimą arba ribokite užklausų dažnį jūsų Zava Writer interneto serveryje
- Sukurkite savo daugiagentę programą jums svarbioje srityje
- Diegkite debesyje pakeisdami Foundry Local į Azure AI Foundry, ta pati kodo bazė, kitoks galinis taškas
