# Deo 13: Radionica završena

> **Čestitamo!** Završili ste celu Foundry Local radionicu.

Prošli ste od instalacije CLI-ja do pravljenja čat aplikacija, RAG pipeline-a, multi-agentnih sistema, transkripcije govora u tekst, kompajliranja sopstvenih prilagođenih modela, omogućavanja pozivanja alata i povezivanja uživo web UI-ja za vašu multi-agentnu aplikaciju. Sve to radi potpuno na vašem uređaju.

---

## Šta ste napravili

| Deo | Šta ste napravili |
|------|------------------|
| 1 | Instalirali Foundry Local, istražili modele kroz CLI |
| 2 | Savladali Foundry Local SDK API za upravljanje servisima, katalogom, kešom i modelima |
| 3 | Povezali se iz Python, JavaScript i C# koristeći SDK sa OpenAI |
| 4 | Napravili RAG pipeline sa lokalnim pretraživanjem znanja |
| 5 | Kreirali AI agente sa ličnostima i strukturiranim izlazom |
| 6 | Orkestrirali multi-agentne pipeline-ove sa povratnim spregama |
| 7 | Istražili produkcionu završnu aplikaciju, Zava Creative Writer |
| 8 | Napravili radne tokove vođene evaluacijom za agente |
| 9 | Transkribovali audio uz pomoć Whisper, govor u tekst na uređaju |
| 10 | Kompajlirali i pokrenuli prilagođeni Hugging Face model sa ONNX Runtime GenAI |
| 11 | Omogućili lokalnim modelima da pozivaju spoljne funkcije pomoću pozivanja alata |
| 12 | Napravili web UI baziran na pregledaču za Zava Creative Writer sa streamingom u realnom vremenu |

---

## Dalje ideje

- Kombinujte pozivanje alata sa agentima za pravljenje autonomnih radnih tokova
- Upitujte lokalnu bazu podataka ili pozivajte interne REST API-je iz vaših alata
- Isprobajte različite modele (`phi-4-mini`, `deepseek-r1-7b`) za poređenje kvaliteta i brzine
- Dodajte autentifikaciju ili ograničenje brzine na vaš Zava Writer web server
- Kreirajte sopstvenu multi-agentnu aplikaciju za oblast koja vas zanima
- Implementirajte u cloud zamenjujući Foundry Local sa Azure AI Foundry, isti kod, drugačiji endpoint
