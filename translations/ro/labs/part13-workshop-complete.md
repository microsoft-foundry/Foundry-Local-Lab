## Idei Suplimentare

- Combină tool calling cu agenți pentru a construi fluxuri autonome de lucru
- Interoghează o bază de date locală sau apelează API-uri REST interne din uneltele tale
- Încearcă modele diferite (`phi-4-mini`, `deepseek-r1-7b`) pentru a compara calitatea și viteza
- Adaugă autentificare sau limitare a ratei pe serverul web Zava Writer
- Creează propria ta aplicație multi-agent pentru un domeniu care te interesează
- Fă deploy în cloud prin înlocuirea Foundry Local cu Azure AI Foundry, același cod, endpoint diferit
