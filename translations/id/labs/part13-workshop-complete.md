# Bagian 13: Lokakarya Selesai

> **Selamat!** Anda telah menyelesaikan seluruh Lokakarya Foundry Local.

Anda telah melalui mulai dari menginstal CLI hingga membangun aplikasi chat, pipeline RAG, sistem multi-agen, transkripsi ucapan-ke-teks, mengompilasi model kustom Anda sendiri, mengaktifkan pemanggilan alat, dan menghubungkan UI web langsung untuk aplikasi multi-agen Anda. Semua itu berjalan sepenuhnya di perangkat Anda.

## Ide Selanjutnya

- Gabungkan pemanggilan alat dengan agen untuk membangun workflow otonom
- Kuery basis data lokal atau panggil REST API internal dari alat Anda
- Coba model berbeda (`phi-4-mini`, `deepseek-r1-7b`) untuk membandingkan kualitas dan kecepatan
- Tambahkan autentikasi atau pembatasan kecepatan pada server web Zava Writer Anda
- Buat aplikasi multi-agen sendiri untuk domain yang Anda pedulikan
- Terapkan ke cloud dengan mengganti Foundry Local dengan Azure AI Foundry, kode sama, titik akhir berbeda
