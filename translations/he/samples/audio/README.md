# קבצי שמע לדוגמה לחלק 7 - תמלול קולי לוויספר

קבצי ה-WAV האלה נוצרים באמצעות `pyttsx3` (טקסט לדיבור של Windows SAPI5) ומתמקדים סביב **מוצרי עשה זאת בעצמך של זאבה** מהדמו של הכותב היוצר.

## יצירת הדגימות

```bash
# מהשורש של המאגר - דורש את .venv עם pyttsx3 מותקן
.venv\Scripts\Activate.ps1          # חלונות
python samples/audio/generate_samples.py
```

## קבצי דוגמה

| קובץ | תרחיש | משך |
|------|----------|----------|
| `zava-customer-inquiry.wav` | לקוח שואל על **מברגת פרוגריפ אלחוטית של זאבה** - מומנט, חיי סוללה, תיק נשיאה | ~15 שניות |
| `zava-product-review.wav` | לקוח נותן ביקורת על **צבע פנים אולטרוסמוט של זאבה** - כיסוי, זמן ייבוש, VOC נמוך | ~22 שניות |
| `zava-support-call.wav` | שיחת תמיכה על **ארגז כלים טיטנלוק של זאבה** - מפתחות חילוף, רפידות למגירות נוספות | ~20 שניות |
| `zava-project-planning.wav` | חובב עשה זאת בעצמך מתכנן דק בחצר האחורית עם **לוח קומפוזיט אקובורד של זאבה** ואורות ברייטבין | ~25 שניות |
| `zava-workshop-setup.wav` | סיור בסדנה מלאה שמשתמשת ב**כל חמשת מוצרי זאבה** | ~32 שניות |
| `zava-full-project-walkthrough.wav` | סיור מרחיב בשיפוץ המוסך הכולל שימוש ב**כל מוצרי זאבה** (למבחני שמע ארוך, ראו [Foundry-Local#517](https://github.com/microsoft/Foundry-Local/issues/517)) | ~4 דקות |

## הערות

- קבצי WAV **מחויבים** למאגר (מיועדים ב). כדי ליצור קבצי .wav חדשים יש להריץ את הסקריפט למעלה ליצירת סקריפטים חדשים או לשנות ליצירת סקריפטים חדשים.
- הסקריפט משתמש בקול **Microsoft David** (אנגלית אמריקאית) במהירות 160 מילים בדקה לתוצאות תמלול ברורות.
- כל התרחישים מתייחסים למוצרים מ-[`zava-creative-writer-local/src/api/agents/writer/products.json`](../../../../zava-creative-writer-local/src/api/agents/writer/products.json).
