# חלק 13: סיום הסדנה

> **מזל טוב!** השלמת את סדנת Foundry Local במלואה.

עברת מהתקנת ה-CLI לבניית יישומי צ'אט, צינורות RAG, מערכות עם סוכנים מרובים, תמלול דיבור לטקסט, הידור מודלים מותאמים אישית, הפעלת קריאת כלים, וחיבור ממשק משתמש מקוון חי ליישום עם סוכנים מרובים שלך. הכל רץ כולו במכשיר שלך.

---

## מה שבנית

| חלק | מה שבנית |
|------|---------------|
| 1 | התקנת Foundry Local, חקר מודלים דרך ה-CLI |
| 2 | שליטה ב-Foundry Local SDK API לשירות, קטלוג, מטמון וניהול מודלים |
| 3 | התחברת מפייתון, JavaScript ו-C# בעזרת ה-SDK ו-OpenAI |
| 4 | בניית צינור RAG עם אחזור ידע מקומי |
| 5 | יצירת סוכני AI עם פרסונות ופלט מובנה |
| 6 | תזמור צינורות עם סוכנים מרובים ולולאות משוב |
| 7 | חקר פיתוח יישום קצה הפקה, זאבה היוצר היצירתי |
| 8 | בניית זרימות עבודה מבוססות הערכה לסוכנים |
| 9 | תמלול קול עם Whisper, דיבור לטקסט במכשיר |
| 10 | הידור והרצת מודל Hugging Face מותאם אישית עם ONNX Runtime GenAI |
| 11 | הפעלת מודלים מקומיים לקרוא לפונקציות חיצוניות עם קריאת כלים |
| 12 | בניית ממשק משתמש בדפדפן לזאבה היוצר היצירתי עם שידור בזמן אמת |

---

## רעיונות נוספים

- לשלב קריאת כלים עם סוכנים לבניית זרימות עבודה אוטונומיות
- לשאול מסד נתונים מקומי או לקרוא API פנימיים מתוך הכלים שלך
- לנסות מודלים שונים (`phi-4-mini`, `deepseek-r1-7b`) להשוואת איכות ומהירות
- להוסיף אימות או הגבלת קצב לשרת הווב של זאבה היוצר שלך
- ליצור יישום עם סוכנים מרובים בתחום שמעניין אותך
- לפרוס לענן על ידי החלפת Foundry Local ב-Azure AI Foundry, אותו קוד, נקודת קצה שונה
