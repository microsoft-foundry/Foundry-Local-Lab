# бһ•бҹ’бһ“бҹӮбһҖ 7бҹ– бһўбҹ’бһ“бһҖбһ“бһ·бһ–бһ“бҹ’бһ’вҖӢбһҹбһ·бһӣбҹ’бһ”бҹҲвҖӢбһҡбһ”бһҹбҹӢ Zava - бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё Capstone

> **бһӮбҹ„бһӣбһ”бҹҶбһҺбһ„бҹ–** бһҹбҹ’бһңбҹӮбһ„бһҷбһӣбҹӢбһўбҹҶбһ–бһёбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёвҖӢбһҳбһ“бһ»бһҹбҹ’бһҹбһ…бҹ’бһҡбһҫбһ“вҖӢбһ”бҹ’бһҡбһ—бҹҒбһ‘вҖӢбһ”бҹӮбһ”бһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳ бһҠбҹӮбһӣбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҠбҹӮбһӣбһҳбһ¶бһ“бһҜбһҖбһ‘бҹҒбһҹбһ”бһҪбһ“бһҡбһјбһ”бһҹбһ бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһӮбҹ’бһ“бһ¶бһҠбһҫбһҳбҹ’бһ”бһёбһ•бһӣбһ·бһҸбһўбһҸбҹ’бһҗбһ”бһ‘бһӮбһ»бһҺбһ—бһ¶бһ–бһ‘бһҹбҹ’бһҹбһ“бһ¶бһңбһҠбҹ’бһҸбһёбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ Zava Retail DIY - бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҮбһ¶бһ‘бһ¶бҹҶбһ„бһҹбҹ’бһҡбһ»бһ„бһӣбһҫбһ§бһ”бһҖбһҡбһҺбҹҚбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҮбһ¶бһҳбһҪбһҷ Foundry Localбҹ”

бһ“бҹҒбҹҮбһӮбһәбһҮбһ¶бһҖбһ¶бһҡбһ”бҹ’бһҡбһЎбһ„ **capstone lab** бһ“бҹғбһңбһӮбҹ’бһӮбһ”бһҺбҹ’бһҠбһ»бҹҮбһ”бһҺбҹ’бһҠбһ¶бһӣбҹ” бһңбһ¶бһ•бҹ’бһӮбһ»бҹҶбһӮбҹ’бһ“бһ¶бһҡбһңбһ¶бһ„бһҡбһҝбһ„бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһҡбҹҖбһ“ - бһҖбһ¶бһҡбһҹбһҳбҹ’бһҡбһ”бһҹбһҳбҹ’бһҡбһҪбһӣ SDK (бһ•бҹ’бһ“бҹӮбһҖ 3), бһҖбһ¶бһҡбһҷбһҖбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ–бһёбһҖбһ“бҹ’бһӣбҹӮбһ„бһҹбҹ’бһҗбһ·бһҸбһҖбҹ’бһ“бһ»бһ„бһҸбҹҶбһ”бһ“бҹӢ (бһ•бҹ’бһ“бҹӮбһҖ 4), бһҡбһјбһ”бһҹбһҳбҹ’бһ”бһҸбҹ’бһҸбһ·бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ (бһ•бҹ’бһ“бҹӮбһҖ 5), бһ“бһ·бһ„бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ…бҹ’бһҡбһҫбһ“ (бһ•бҹ’бһ“бҹӮбһҖ 6) - бһ…бһјбһӣбһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ–бҹҒбһүбһӣбҹҒбһүбһҠбҹӮбһӣбһҳбһ¶бһ“бһҖбҹ’бһ“бһ»бһ„ **Python**, **JavaScript**, бһ“бһ·бһ„ **C#** бҹ”

### бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ”бһ“бҹ’бһ‘бһ¶бһҸбҹӢбһ”бһҺбҹ’бһҠбҹ„бһҷ

1. **бһўбҹ’бһ“бһҖбһҹбҹ’бһҡбһ¶бһңбһҮбҹ’бһҡбһ¶бһң** бһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһ”бҹ’бһҡбһ’бһ¶бһ“бһ”бһ‘ бһ“бһ·бһ„бһ”бһ„бҹ’бһҖбһҫбһҸбһҖбҹҶбһҺбһҸбҹӢбһҸбҹ’бһҡбһ¶бһҹбҹ’бһҡбһ¶бһңбһҮбҹ’бһҡбһ¶бһңбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’ (JSON)
2. **бһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһ•бһӣбһ·бһҸбһ•бһӣ** бһҹбһҪбһҡбһҹбҹҶбһҺбһҪбһҡбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҸбһ¶бһЎбһ»бһҖбһ•бһӣбһ·бһҸбһ•бһӣбһҖбҹ’бһ“бһ»бһ„бһҸбҹҶбһ”бһ“бҹӢбһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһ–бһ¶бһҖбҹ’бһҷбһҹбҹҶбһҺбһҪбһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһҠбҹ„бһҷ LLM
3. **бһўбҹ’бһ“бһҖбһ“бһ·бһ–бһ“бҹ’бһ’** бһ”бҹ’бһҡбһҳбһјбһӣбһҹбҹ’бһҡбһ¶бһңбһҮбҹ’бһҡбһ¶бһң + бһ•бһӣбһ·бһҸбһ•бһӣ + бһҖбһ·бһ…бҹ’бһ…бһҖбһ¶бһҡбһҮбһјбһ“бһ‘бҹ…бһҮбһ¶бһўбһҸбҹ’бһҗбһ”бһ‘бһ”бһ“бҹ’бһҸ бһ…бһҳбҹ’бһӣбһ„бһҳбһҸбһ·бһҷбҹ„бһ”бһӣбҹӢбһ”бһ“бҹ’бһҗбҹӮбһҳбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһ”бҹ’бһҡбһҫ `---`
4. **бһўбҹ’бһ“бһҖбһҖбҹӮбһҹбһҳбҹ’бһҡбһҪбһӣ** бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһўбһҸбҹ’бһҗбһ”бһ‘ бһ“бһ·бһ„бһҸбһ”бһҸбһ”бһҮбһјбһ“бһҳбһҸбһ·бһҷбҹ„бһ”бһӣбҹӢбһҮбһ¶бһ‘бһҳбҹ’бһҡбһ„бҹӢ JSONбҹ–
   - `"accept"` вҶ’ бһ”бһ“бҹ’бһ‘бһ¶бһҸбҹӢбһ”бһҺбҹ’бһҠбҹ„бһҷвҖӢбһ”бһүбҹ’бһ…бһ”бҹӢ
   - `"revise"` вҶ’ бһҳбһҸбһ·бһҷбҹ„бһ”бһӣбҹӢбһҸбҹ’бһҡбһјбһңбһ•бҹ’бһүбһҫбһҸбҹ’бһҡбһЎбһ”бҹӢбһ‘бҹ…бһўбҹ’бһ“бһҖбһҹбҹ’бһҡбһ¶бһңбһҮбҹ’бһҡбһ¶бһң бһ“бһ·бһ„бһўбҹ’бһ“бһҖбһ“бһ·бһ–бһ“бҹ’бһ’ (бһ…бҹҶбһ“бһҪбһ“ 2 бһҠбһ„бһ”бҹҶбһ•бһ»бһҸ)

### бһңбһ·бһ’бһёбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„ 3 - бһҸбһ¶бһҳбһҠбһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҡбһҪбһҳ

бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһҖбҹ’бһ“бһ»бһ„бһ”бһ“бҹ’бһ‘бһ¶бһҸбҹӢбһ”бһҺбҹ’бһҠбҹ„бһҷбһ…бҹӮбһҖбһ…бһ¶бһҷбһўбҹ’бһ“бһҖбһўбһҸбһ·бһҗбһ·бһҮбһ“бһҳбҹүбһјбһҠбҹӮбһӣ Foundry Local бһҸбҹӮбһҳбһҪбһҷбҹ” бһҹбһ·бһҖбҹ’бһҹбһ¶бһңбһ·бһ’бһёбһҠбҹҶбһЎбһҫбһ„бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ—бһ¶бһҹбһ¶бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбҹ–

<details>
<summary><strong>рҹҗҚ Python - foundry_config.py</strong></summary>

```python
from foundry_local import FoundryLocalManager

MODEL_ALIAS = "phi-3.5-mini"

# бһҮбҹҶбһ бһ¶бһ“бһ‘бһё бҹЎбҹ– бһ”бһ„бҹ’бһҖбһҫбһҸбһўбҹ’бһ“бһҖбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„ бһ“бһ·бһ„бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳ Foundry Local
manager = FoundryLocalManager()
manager.start_service()

# бһҮбҹҶбһ бһ¶бһ“бһ‘бһё бҹўбҹ– бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҳбһҫбһӣбһҗбһ¶бһҸбһҫбһҳбҹүбһјбһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ‘бһ¶бһүбһҷбһҖбһҡбһҪбһ…бһ бһҫбһҷбһ¬бһ“бҹ…
cached = manager.list_cached_models()
catalog_info = manager.get_model_info(MODEL_ALIAS)
is_cached = any(m.id == catalog_info.id for m in cached) if catalog_info else False

if not is_cached:
    manager.download_model(MODEL_ALIAS)

# бһҮбҹҶбһ бһ¶бһ“бһ‘бһё бҹЈбҹ– бһ•бҹ’бһ‘бһ»бһҖбһҳбҹүбһјбһҠбҹӮбһӣбһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„бһўбһҸбҹӢбһ…бһ¶бҹҶ
manager.load_model(MODEL_ALIAS)
model_id = manager.get_model_info(MODEL_ALIAS).id

# бһҶбҹ’бһӣбһҫбһҷбһҸбһ” OpenAI бһҠбҹӮбһӣбһ…бҹӮбһҖбһҡбҹҶбһӣбҹӮбһҖ
client = openai.OpenAI(base_url=manager.endpoint, api_key=manager.api_key)
```

бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһ“бһ¶бҹҶбһ…бһјбһӣ `from foundry_config import client, model_id` бҹ”

</details>

```javascript
import { FoundryLocalManager } from "foundry-local-sdk";
import { OpenAI } from "openai";

FoundryLocalManager.create({ appName: "ZavaCreativeWriter" });
const manager = FoundryLocalManager.instance;
await manager.startWebService();

// бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҹбҹ’бһҸбһ»бһҖбһ–бһҸбҹҢбһҳбһ¶бһ“ вҶ’ бһ‘бһ¶бһүбһҷбһҖ вҶ’ бһ•бҹ’бһ‘бһ»бһҖ (бһҡбһ…бһ“бһ¶бһ”бҹҗбһ‘бҹ’бһҳ SDK бһҗбҹ’бһҳбһё)
const catalog = manager.catalog;
const model = await catalog.getModel(MODEL_ALIAS);
if (!model.isCached) {
  console.log(`Downloading model: ${MODEL_ALIAS}...`);
  await model.download();
}
await model.load();

const client = new OpenAI({ baseURL: manager.urls[0] + "/v1", apiKey: "foundry-local" });
const modelId = model.id;
export { client, modelId };
```

### бһңбһ·бһ’бһёбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„ 4 - бһҷбһӣбҹӢбһҠбһ№бһ„бһ–бһёбһҡбһ„бҹ’бһңбһ·бһӣбһҳбһҸбһ·бһҷбҹ„бһ”бһӣбҹӢ

бһҡбһ„бҹ’бһңбһ·бһӣбһҳбһҸбһ·бһҷбҹ„бһ”бһӣбҹӢбһӮбһәбһҮбһ¶бһўбҹ’бһңбһёбһҠбҹӮбһӣбһ’бҹ’бһңбһҫбһўбҹ„бһҷбһ”бһ“бҹ’бһ‘бһ¶бһҸбҹӢбһ”бһҺбҹ’бһҠбҹ„бһҷбһ“бҹҒбҹҮ вҖңбһҶбҹ’бһӣбһ¶бһҸвҖқ вҖ“ бһўбҹ’бһ“бһҖбһҖбҹӮбһҹбһҳбҹ’бһҡбһҪбһӣбһўбһ¶бһ…бһ”бһүбҹ’бһҮбһјбһ“бһҖбһ¶бһҡбһ„бһ¶бһҡбһҸбҹ’бһҡбһЎбһ”бҹӢбһҳбһҖбһңбһ·бһүбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҖбҹӮбһ”бҹ’бһҡбҹӮбҹ” бһҸбһ¶бһҳбһҠбһ¶бһ“бһ бҹҒбһҸбһ»бһ•бһӣбҹ–

```
Orchestrator:
  1. researcher.research(topic, "No Feedback")    вҶҗ first pass
  2. product.findProducts(productContext)
  3. writer.write(research, products, assignment)  вҶҗ streams article
  4. Split article at "---" вҶ’ article + writerFeedback
  5. editor.edit(article, writerFeedback)

  WHILE editor says "revise" AND retryCount < 2:
    6. researcher.research(topic, editor.researchFeedback)  вҶҗ refined
    7. writer.write(research, products, editor.editorFeedback)
    8. editor.edit(newArticle, newWriterFeedback)
    9. retryCount++
```

**бһҹбҹҶбһҺбһҪбһҡбһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһ–бһ·бһ…бһ¶бһҡбһҺбһ¶бҹ–**
- бһ бҹҒбһҸбһ»бһўбҹ’бһңбһёбһ”бһ¶бһ“бһҮбһ¶бһ…бҹҶбһ“бһҪбһ“бһ–бҹ’бһҷбһ¶бһҷбһ¶бһҳбһҮбһҪбһҹбһҮбһ»бһӣбһҖбҹҶбһҺбһҸбҹӢбһҸбҹ’бһҡбһ№бһҳ 2? бһҸбһҫбһҳбһ¶бһ“бһўбҹ’бһңбһёбһҖбһҫбһҸбһЎбһҫбһ„ бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһ“бһҖбһ”бһ“бҹ’бһҸбһ”бһ“бҹ’бһҗбҹӮбһҳ?
- бһ бҹҒбһҸбһ»бһўбҹ’бһңбһёбһ”бһ¶бһ“бһҮбһ¶бһўбҹ’бһ“бһҖбһҹбҹ’бһҡбһ¶бһңбһҮбҹ’бһҡбһ¶бһңбһ‘бһ‘бһҪбһӣ `researchFeedback` бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһўбҹ’бһ“бһҖбһ“бһ·бһ–бһ“бҹ’бһ’бһ‘бһ‘бһҪбһӣ `editorFeedback`?
- бһҸбһҫбһҳбһ¶бһ“бһўбҹ’бһңбһёбһҖбһҫбһҸбһЎбһҫбһ„ бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһ“бһҖбһҖбҹӮбһҹбһҳбҹ’бһҡбһҪбһӣбһҸбҹӮбһ„бһҸбҹӮбһ”бһүбҹ’бһҮбһ¶бһҖбҹӢ "revise"?

## бһҮбҹҶбһ бһ¶бһ“бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢ

бһ”бһ“бҹ’бһҸбһ‘бҹ… [бһ•бҹ’бһ“бҹӮбһҖ 8бҹ– бһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚбһҠбҹ„бһҷбһҷбһҖбһҖбһ¶бһҡбһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғбһҮбһ¶бһҳбһҪбһӣбһҠбҹ’бһӢбһ¶бһ“](part8-evaluation-led-development.md) бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһҸбһҹбҹҶбһҺбһ»бҹҶбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһҜбһҖбһҹбһ¶бһҡбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҳбһ¶бһҹ бһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһ…бҹ’бһ”бһ¶бһ”бҹӢ бһ“бһ·бһ„бһҖбһ¶бһҡбһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғ LLM бһҮбһ¶бһўбҹ’бһ“бһҖбһңбһ¶бһҷбһҸбҹҶбһӣбҹғбҹ”
