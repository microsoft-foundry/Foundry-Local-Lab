### бһҖбһ·бһ…бҹ’бһ…бһҖбһ¶бһҡ 2 - бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҡбһ”бһҹбҹӢбһ”бҹүбһ¶бһҷбһ”бҹҠбһёбһ“

бһҹбһ·бһҖбҹ’бһҹбһ¶бһҗбһ¶бһҸбһҫбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢ бһ“бһ·бһ„бһ—бҹ’бһҮбһ¶бһ”бҹӢбһҷбҹүбһ¶бһ„бһҠбһјбһ…бһҳбҹ’бһҸбҹҒбһ…бҹ–

**1. бһўбһҸбһ·бһҗбһ·бһҮбһ“бһҳбҹүбһјбһҠбҹӮбһӣбһҡбһҪбһҳ**

бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһ…бҹӮбһҖбһҡбҹҶбһӣбҹӮбһҖбһҳбҹүбһјбһҠбҹӮбһӣ Foundry Local бһҸбҹӮбһҳбһҪбһҷбҹ–

**2. бһҖбһ¶бһҡбһҺбҹӮбһ“бһ¶бҹҶбһ–бһ·бһҹбҹҒбһҹ**

бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҡбҹҖбһ„бһҡбһ¶бһӣбҹӢбһҸбҹ’бһҡбһҪбһҸбһ”бһ¶бһүбҹӢбһҳбһ¶бһ“бһ”бһ»бһӮбҹ’бһӮбһӣбһ·бһҖбһӣбһҖбҹ’бһҒбһҺбҹҲбһҒбһ»бһҹбһӮбҹ’бһ“бһ¶бҹ–

| бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ | бһҖбһ¶бһҡбһҺбҹӮбһ“бһ¶бҹҶ (бһҹбһ„бҹ’бһҒбҹҒбһ”) |
|-------|----------------------|
| Researcher | "бһ•бҹ’бһҸбһӣбҹӢбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһҹбҹҶбһҒбһ¶бһ“бҹӢбҹ— бһҹбҹ’бһҗбһ·бһҸбһ· бһ“бһ·бһ„бһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһ•бҹ’бһ“бҹӮбһҖбҹ” бһҡбҹҖбһ”бһ…бҹҶбһҮбһ¶бһ…бҹҶбһҺбһ»бһ…бһӮбҹ„бһӣбҹ”" |
| Writer | "бһҹбһҡбһҹбҹҒбһҡбһўбһҸбҹ’бһҗбһ”бһ‘бһ”бҹ’бһӣбһҖбҹӢбһ‘бһ¶бһҖбҹӢбһ‘бһ¶бһү (3-4 бһ…бҹҶбһҺбһ»бһ…бһҖбһҗбһ¶) бһ–бһёбһҖбҹҶбһҺбһҸбҹӢбһҸбҹ’бһҡбһ¶бһҹбҹ’бһҡбһ¶бһңбһҮбҹ’бһҡбһ¶бһңбҹ” бһҳбһ·бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһҗбҹ’бһҳбһёбҹ”" |
| Editor | "бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһ…бҹ’бһ”бһ¶бһҹбҹӢбһӣбһ¶бһҹбҹӢ бһҹбҹҶбһҺбһ»бҹҶбһҡбһҝбһ„ бһ“бһ·бһ„бһ—бһ¶бһ–бһҹбһҳбһҡбһҳбҹ’бһҷбһ“бҹғбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бҹ” бһҹбҹҒбһ…бһҖбҹ’бһҠбһёбһҹбһҳбҹ’бһҡбҹҒбһ…бҹ– бһ‘бһ‘бһҪбһӣбһҷбһҖ бһ¬ бһҖбҹӮбһ”бҹ’бһҡбҹӮбҹ”" |

**3. бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһңбһ·бһүбһҡбһңбһ¶бһ„бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ**


бһ“бҹҒбҹҮбһҮбһ¶бһҡбһ…бһ“бһ¶бһ”бҹҗбһ‘бҹ’бһҳбһҖбһ¶бһҡбһ…бһ„бһҖбҹ’бһҡбһ„бһҠбҹӮбһӣбһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһ”бһ¶бһ“бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ–бһ бһ» (бһ”бһ¶бһ“бһҹбһ·бһҖбҹ’бһҹбһ¶вҖӢбһҮбҹ’бһҡбһ¶бһӣбһҮбҹ’бһҡбҹ…бһ“бҹ… [бһ•бҹ’бһ“бҹӮбһҖ 7](part7-zava-creative-writer.md))бҹ–

### бһ”бҹүбһ¶бһҷбһ”бҹҠбһёбһ“бһҸбһ¶бһҳбһӣбҹҶбһҠбһ¶бһ”бҹӢ

![Sequential Pipeline](../../../translated_images/km/part6-sequential-pipeline.1c0bf50e3da3fe47.webp)

бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ“бһёбһҳбһҪбһҷбҹ—бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ•бһӣбһ·бһҸбһ•бһӣбһҡбһ”бһҹбҹӢбһҳбһ»бһҒбһ„бһ¶бһҡбһҳбһ»бһ“бҹ” бһ”бҹ’бһҡбҹӮбһ”бҹ’бһҡбһҪбһӣбһӣбһҫбһҹбһ¶бһҳбһүбҹ’бһүбһ“бһ·бһ„бһўбһ¶бһ…бһ‘бһ¶бһҷбһ‘бҹ’бһҡбһ„бҹӢбһ”бһ¶бһ“бҹ”

### бһ…бҹ’бһҡбһҖбһҹбҹ’бһҸбһ»бһҖбһҳбһҸбһ·

![Feedback Loop](../../../translated_images/km/part6-feedback-loop.40f9ce0aa4b8ae78.webp)

бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғбһўбһ¶бһ…бһ”бһүбҹ’бһ…бһјбһӣбһ…бҹҶбһҺбһ»бһ…бһІбҹ’бһҷбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһңбһ·бһүбһ“бҹ…бһҮбҹҶбһ бһ¶бһ“бһҳбһ»бһ“бҹ—бҹ” бһўбҹ’бһ“бһҖбһҹбһҡбһҹбҹҒбһҡ Zava бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҖбһ¶бһҡбһ“бҹҒбҹҮбҹ– бһўбҹ’бһ“бһҖбһҖбҹӮбһҹбһҳбҹ’бһҡбһҪбһӣбһўбһ¶бһ…бһ•бҹ’бһүбһҫбһҳбһҸбһ·бһңбһ·бһүбһ‘бҹ…бһҖбһ¶бһ“бҹӢбһўбҹ’бһ“бһҖбһҹбҹ’бһҡбһ¶бһңбһҮбҹ’бһҡбһ¶бһң бһ“бһ·бһ„бһўбҹ’бһ“бһҖбһҹбһҡбһҹбҹҒбһҡбҹ”

### бһ”бһҡбһ·бһ”бһ‘бһҡбһҪбһҳ

![Shared Context](../../../translated_images/km/part6-shared-context.63704188ab4995e4.webp)

бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһ…бҹӮбһҖбһҡбҹҶбһӣбҹӮбһҖ `foundry_config` бһҠбһјбһ…бһӮбҹ’бһ“бһ¶бһ”бҹ’бһҡбһҫбһҳбҹүбһјбһҠбҹӮбһӣбһ“бһ·бһ„бһ…бҹҶбһҺбһ»бһ…бһ…бһ»бһ„бҹ”

|---------|-----------------|
| бһҖбһ¶бһҡбһ–бһ·бһҹбҹҒбһҹбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ | бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ“бһёбһҳбһҪбһҷбҹ—бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҡбһҝбһ„бһҳбһҪбһҷбһҮбһ¶бһҳбһҪбһҷбһҖбһ¶бһҡбһҺбҹӮбһ“бһ¶бҹҶбһ•бҹ’бһҸбҹ„бһҸбһӣбһҫ |
| бһҖбһ¶бһҡбһ”бһүбҹ’бһҮбһјбһ“бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ | бһӣбһ‘бҹ’бһ’бһ•бһӣбһ–бһёбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҳбһҪбһҷбһҖбҹ’бһӣбһ¶бһҷбһҮбһ¶ бһ§бһ”бһҖбһҡбһҺбҹҚбһ”бһүбҹ’бһ…бһјбһӣбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢ |
| бһ…бҹ’бһҡбһҖбһҹбҹ’бһҸбһ»бһҖбһҳбһҸбһ· | бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғбһўбһ¶бһ…бһ”бһ„бҹ’бһ бһ¶бһүбһҖбһ¶бһҡбһҮбһҝбһ“бһӣбһҝбһ“ бһҠбһҫбһҳбҹ’бһ”бһёбһӮбһ»бһҺбһ—бһ¶бһ–бһҒбҹ’бһ–бһҹбҹӢаёӮаё¶а№үаёҷ |
| бһӣбһ‘бҹ’бһ’бһ•бһӣбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’ | бһ…бһҳбҹ’бһӣбһҫбһҷбһҡбҹҒбһ бҹ’бһңбҹҗбһҡбһҳбҹҒ JSON бһ•бҹ’бһҸбһӣбҹӢбһ“бһјбһңбһҖбһ¶бһҡбһ”бҹ’бһҡбһ¶бһҹбҹ’бһҡбҹҗбһҷбһ‘бһ¶бһҖбҹӢбһ‘бһ„бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҷбҹүбһ¶бһ„бһ‘бһ»бһҖбһ…бһ·бһҸбҹ’бһҸбһ”бһ¶бһ“ |
| бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„ | бһўбҹ’бһ“бһҖбһҹбһҳбҹ’бһҡбһ”бһҹбһҳбҹ’бһҡбһҪбһӣбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһӣбҹҶбһҠбһ¶бһ”бҹӢбһ”бҹүбһ¶бһҷбһ”бҹҠбһёбһ“ бһ“бһ·бһ„бһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһҖбҹҶбһ бһ»бһҹ |
