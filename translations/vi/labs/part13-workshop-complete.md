# Phần 13: Hoàn thành Workshop

> **Chúc mừng bạn!** Bạn đã hoàn thành toàn bộ Workshop Foundry Local.

Bạn đã đi từ việc cài đặt CLI đến xây dựng các ứng dụng trò chuyện, pipeline RAG, hệ thống đa tác nhân, phiên âm giọng nói thành văn bản, biên dịch các mô hình tùy chỉnh của riêng bạn, kích hoạt gọi công cụ và kết nối giao diện web trực tiếp cho ứng dụng đa tác nhân của bạn. Tất cả đều chạy hoàn toàn trên thiết bị của bạn.

---

## Những gì bạn đã xây dựng

| Phần | Những gì bạn đã xây dựng |
|------|-------------------------|
| 1 | Cài đặt Foundry Local, khám phá các mô hình qua CLI |
| 2 | Thành thạo API Foundry Local SDK cho dịch vụ, danh mục, bộ đệm, và quản lý mô hình |
| 3 | Kết nối từ Python, JavaScript, và C# sử dụng SDK với OpenAI |
| 4 | Xây dựng pipeline RAG với truy xuất kiến thức cục bộ |
| 5 | Tạo các tác nhân AI với personas và đầu ra có cấu trúc |
| 6 | Điều phối pipeline đa tác nhân với vòng phản hồi |
| 7 | Khám phá ứng dụng sản xuất đỉnh cao, Zava Creative Writer |
| 8 | Xây dựng quy trình phát triển dựa trên đánh giá cho các tác nhân |
| 9 | Phiên âm âm thanh với Whisper, chuyển giọng nói thành văn bản trên thiết bị |
| 10 | Biên dịch và chạy mô hình Hugging Face tùy chỉnh với ONNX Runtime GenAI |
| 11 | Cho phép mô hình cục bộ gọi các hàm bên ngoài với công cụ gọi |
| 12 | Xây dựng giao diện người dùng trên trình duyệt cho Zava Creative Writer với streaming thời gian thực |

---

## Ý tưởng tiếp theo

- Kết hợp công cụ gọi với các tác nhân để xây dựng quy trình làm việc tự động
- Truy vấn cơ sở dữ liệu cục bộ hoặc gọi các REST API nội bộ từ công cụ của bạn
- Thử các mô hình khác nhau (`phi-4-mini`, `deepseek-r1-7b`) để so sánh chất lượng và tốc độ
- Thêm xác thực hoặc giới hạn tần suất vào máy chủ web Zava Writer của bạn
- Tạo ứng dụng đa tác nhân riêng cho lĩnh vực bạn quan tâm
- Triển khai lên đám mây bằng cách thay thế Foundry Local bằng Azure AI Foundry, cùng mã nguồn, điểm cuối khác
