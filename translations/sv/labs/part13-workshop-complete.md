# Del 13: Workshop Klar

> **Grattis!** Du har slutfört hela Foundry Local Workshop.

Du har gått från att installera CLI till att bygga chattapplikationer, RAG-pipelines, multi-agent system, tal-till-text transkription, kompilering av egna anpassade modeller, aktivering av verktygsanrop och koppling av ett live web UI för din multi-agentapplikation. Allt detta körs helt på din enhet.

## Ytterligare idéer

- Kombinera verktygsanrop med agenter för att bygga autonoma arbetsflöden
- Fråga en lokal databas eller anropa interna REST-API:er från dina verktyg
- Prova olika modeller (`phi-4-mini`, `deepseek-r1-7b`) för att jämföra kvalitet och hastighet
- Lägg till autentisering eller hastighetsbegränsning till din Zava Writer webbserver
- Skapa din egen multi-agentapplikation för ett område du bryr dig om
- Distribuera till molnet genom att byta ut Foundry Local mot Azure AI Foundry, samma kod, annan endpoint
