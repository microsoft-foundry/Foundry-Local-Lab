# حصہ 13: ورکشاپ مکمل

> **مبارک ہو!** آپ نے مکمل Foundry Local ورکشاپ مکمل کر لی ہے۔

آپ نے CLI کی تنصیب سے لے کر چیٹ ایپلیکیشنز بنانے، RAG پائپ لائنز، ملٹی ایجنٹ سسٹمز، اسپیچ ٹو ٹیکسٹ ٹرانسکرپشن، اپنی مرضی کے ماڈلز کمپائل کرنے، ٹول کالنگ کو فعال کرنے، اور اپنے ملٹی ایجنٹ ایپلیکیشن کے لیے ایک لائیو ویب UI بنانے تک سب کچھ کیا ہے۔ یہ سب کچھ مکمل طور پر آپ کے آلے پر چل رہا ہے۔

---

## آپ نے کیا بنایا

| حصہ | آپ نے کیا بنایا |
|------|-----------------|
| 1 | Foundry Local انسٹال کیا، CLI کے ذریعے ماڈلز کو دریافت کیا |
| 2 | Foundry Local SDK API کو سروس، کیٹلاگ، کیش، اور ماڈل مینجمنٹ کے لیے ماہر بنایا |
| 3 | SDK کے ذریعے Python، JavaScript، اور C# سے OpenAI کے ساتھ کنیکٹ کیا |
| 4 | لوکل نالج ریٹریول کے ساتھ ایک RAG پائپ لائن بنائی |
| 5 | شخصیات اور ساختہ آؤٹ پٹ کے ساتھ AI ایجنٹس بنائے |
| 6 | فیڈ بیک لوپس کے ساتھ ملٹی ایجنٹ پائپ لائنز کا انتظام کیا |
| 7 | ایک پروڈکشن کیپ اسٹون ایپلیکیشن، Zava Creative Writer کا جائزہ لیا |
| 8 | ایجنٹس کے لیے ایویلیوایشن-لیڈ ڈیولپمنٹ ورک فلو تیار کیے |
| 9 | Whisper کے ذریعے آڈیو کی ٹرانسکرپشن کی، ڈیوائس پر اسپیچ ٹو ٹیکسٹ کی خصوصیت استعمال کی |
| 10 | ONNX Runtime GenAI کے ساتھ ایک حسب ضرورت Hugging Face ماڈل کمپائل اور رن کیا |
| 11 | ٹول کالنگ کے ساتھ لوکل ماڈلز کو بیرونی فنکشنز کال کرنے کے قابل بنایا |
| 12 | Zava Creative Writer کے لیے ایک براؤزر بیسڈ UI بنایا جو ریئل ٹائم اسٹریمنگ فراہم کرتا ہے |

---

## مزید خیالات

- ایجنٹس کے ساتھ ٹول کالنگ کو ملا کر خودکار ورک فلو بنائیں
- اپنے ٹولز سے لوکل ڈیٹابیس یا اندرونی REST APIs کو سوال کریں
- معیار اور رفتار کا موازنہ کرنے کے لیے مختلف ماڈلز (`phi-4-mini`, `deepseek-r1-7b`) آزما کر دیکھیں
- اپنے Zava Writer ویب سرور میں توثیق یا ریٹ لمٹنگ شامل کریں
- اپنے دلچسپی کے شعبے کے لیے اپنی ملٹی ایجنٹ ایپلیکیشن تیار کریں
- Foundry Local کو Azure AI Foundry کے ساتھ تبدیل کر کے کلاؤڈ پر ڈپلائے کریں، کوڈ ایک جیسا، صرف اینڈپوائنٹ مختلف
