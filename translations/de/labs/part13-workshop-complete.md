# Teil 13: Workshop abgeschlossen

> **Herzlichen Glückwunsch!** Sie haben den vollständigen Foundry Local Workshop abgeschlossen.

Sie sind vom Installieren der CLI über das Erstellen von Chat-Anwendungen, RAG-Pipelines, Multi-Agenten-Systemen, Sprach-zu-Text-Transkriptionen, dem Kompilieren eigener benutzerdefinierter Modelle, dem Aktivieren von Tool-Aufrufen bis hin zum Verbinden einer Live-Web-UI für Ihre Multi-Agenten-Anwendung gegangen. Alles läuft vollständig auf Ihrem Gerät.

## Weitere Ideen

- Kombinieren Sie Tool Calling mit Agenten, um autonome Workflows zu erstellen
- Abfragen einer lokalen Datenbank oder Aufrufen interner REST-APIs über Ihre Tools
- Verschiedene Modelle ausprobieren (`phi-4-mini`, `deepseek-r1-7b`), um Qualität und Geschwindigkeit zu vergleichen
- Authentifizierung oder Rate Limiting für Ihren Zava Writer Webserver hinzufügen
- Erstellen Sie Ihre eigene Multi-Agenten-Anwendung für ein Thema, das Ihnen wichtig ist
- Deployment in der Cloud durch Austausch von Foundry Local gegen Azure AI Foundry, gleicher Code, anderer Endpunkt
