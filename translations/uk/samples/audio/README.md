# Зразки аудіофайлів для Частини 7 - Транскрипція голосу Whisper

Ці WAV-файли створені за допомогою `pyttsx3` (Windows SAPI5 текст-в-речь) і тематизовані навколо **продуктів Zava DIY** з демонстрації Creative Writer.

## Створення зразків

```bash
# Від кореня сховища - необхідна .venv з встановленим pyttsx3
.venv\Scripts\Activate.ps1          # Windows
python samples/audio/generate_samples.py
```

## Зразки файлів

| Файл | Сценарій | Тривалість |
|------|----------|------------|
| `zava-customer-inquiry.wav` | Клієнт запитує про **Zava ProGrip Cordless Drill** - крутний момент, тривалість батареї, чохол для перенесення | ~15 сек |
| `zava-product-review.wav` | Огляд клієнтом **Zava UltraSmooth Interior Paint** - покриття, час висихання, низький рівень ЛОС | ~22 сек |
| `zava-support-call.wav` | Дзвінок в підтримку щодо **Zava TitanLock Tool Chest** - заміна ключів, додаткові підкладки для ящиків | ~20 сек |
| `zava-project-planning.wav` | Планування майстром задньої тераси з **Zava EcoBoard Composite Decking** та світильниками BrightBeam | ~25 сек |
| `zava-workshop-setup.wav` | Огляд повністю обладнаної майстерні з використанням **всіх п’яти продуктів Zava** | ~32 сек |
| `zava-full-project-walkthrough.wav` | Детальний огляд ремонту гаража з використанням **всіх продуктів Zava** (для тестування довгого аудіо див. [Foundry-Local#517](https://github.com/microsoft/Foundry-Local/issues/517)) | ~4 хв |

## Примітки

- WAV-файли **зафіксовані** в репозиторії (перелічені в `. Для створення нових .wav-файлів запустіть скрипт вище для генерації нових скриптів або змініть його для створення нових версій.
- Скрипт використовує голос **Microsoft David** (американська англійська) зі швидкістю 160 слів на хвилину для чітких результатів транскрипції.
- Усі сценарії посилаються на продукти з файлу [`zava-creative-writer-local/src/api/agents/writer/products.json`](../../../../zava-creative-writer-local/src/api/agents/writer/products.json).
