# Частина 13: Майстерня завершена

> **Вітаємо!** Ви пройшли повний майстер-клас Foundry Local.

Ви пройшли шлях від встановлення CLI до створення чат-додатків, RAG конвеєрів, багатофункціональних агентських систем, транскрипції мови в текст, компіляції власних моделей, активації виклику інструментів і підключення живого веб-інтерфейсу для вашого багатофункціонального агента. Все це працює повністю на вашому пристрої.

## Подальші ідеї

- Поєднувати виклики інструментів з агентами для побудови автономних робочих процесів
- Запитувати локальну базу даних або викликати внутрішні REST API з ваших інструментів
- Спробувати різні моделі (`phi-4-mini`, `deepseek-r1-7b`) для порівняння якості та швидкості
- Додати автентифікацію або обмеження швидкості до вашого веб-сервера Zava Writer
- Створити власний багатофункціональний додаток для предметної області, яка вас цікавить
- Розгорнути в хмарі, замінивши Foundry Local на Azure AI Foundry, той самий код, інша кінцева точка
