# Část 13: Workshop dokončen

> **Gratulujeme!** Úplně jste dokončili Foundry Local Workshop.

Prošli jste od instalace CLI přes vytváření chatovacích aplikací, RAG pipeline, víceagentních systémů, přepisu řeči na text, kompilace vlastních modelů, povolení volání nástrojů až po propojení živého webového rozhraní pro vaši víceagentní aplikaci. Vše běžící zcela na vašem zařízení.

---

## Co jste vytvořili

| Část | Co jste vytvořili |
|------|-------------------|
| 1 | Nainstalovali Foundry Local, prozkoumali modely přes CLI |
| 2 | Ovládli Foundry Local SDK API pro správu služeb, katalogu, cache a modelů |

---

## Další nápady

- Kombinujte tool calling s agenty pro vytváření autonomních pracovních postupů
- Dotazujte se do lokální databáze nebo volejte interní REST API z vašich nástrojů
- Vyzkoušejte různé modely (`phi-4-mini`, `deepseek-r1-7b`) pro porovnání kvality a rychlosti
- Přidejte autentifikaci nebo omezení rychlosti na váš webový server Zava Writer
- Vytvořte si vlastní víceagentní aplikaci pro obor, který vás zajímá
- Deployněte do cloudu výměnou Foundry Local za Azure AI Foundry, stejný kód, jiný endpoint
