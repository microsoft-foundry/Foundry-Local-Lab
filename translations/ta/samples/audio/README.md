# பகுதி 7 - விச்பர் குரல் உருமாற்றத்திற்கான மாதிரி ஒலி கோப்புகள்

இந்த WAV கோப்புகள் `pyttsx3` (Windows SAPI5 எழுத்து-மொழி மாற்றி) பயன்படுத்தி உருவாக்கப்பட்டுள்ளன மற்றும் Creative Writer டெமோவில் இருந்து **Zava DIY பொருட்கள்** மூலம் தீம் செய்யப்பட்டுள்ளது.

## மாதிரிகளை உருவாக்குக

```bash
# ரெப்போ ரூட்டிலிருந்து - pyttsx3 நிறுவப்பட்ட .venv தேவை
.venv\Scripts\Activate.ps1          # விண்டோசு
python samples/audio/generate_samples.py
```

## குறிப்புகள்

- WAV கோப்புகள் **கிடைக்ககப்படுகின்றன** ( `.wav` கோப்புகள் உருவாக்க மேல் கொடுக்கப்பட்ட உருப்படி இயக்கி புதிய ஸ்கிரிப்ட்களை உருவாக்கலாம் அல்லது மாற்றலாம்).
- ஸ்கிரிப்ட் 160 WPM வேகத்தில் தெளிவான உருமாற்ற முடிவுகளுக்காக **Microsoft David** (அமெரிக்க ஆங்கிலம்) குரலை பயன்படுத்துகிறது.
- அனைத்து சூழ்ச்சிகளும் [`zava-creative-writer-local/src/api/agents/writer/products.json`](../../../../zava-creative-writer-local/src/api/agents/writer/products.json) இல் உள்ள பொருட்களை குறிப்பிடுகின்றன.
