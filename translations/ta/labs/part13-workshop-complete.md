# பகுதி 13: பணிமனை முடிந்தது

> **நல்வாழ்த்துகள்!** நீங்கள் முழு Foundry Local பணிமனையை முடித்துவிட்டீர்கள்.

நீங்கள் CLI ஐ நிறுவுவதிலிருந்து கொண்டு, சந்தை பயன்பாடுகள் உருவாக்குதல், RAG குழாய்முறை, பன்முக செயற்குழுக்கள், பேச்சு-உரை மாற்றம், உங்கள் தனிப்பயன் மாதிரிகள் தொகுப்பது, கருவி அழைப்பை இயக்கு, மற்றும் உங்கள் பன்முக செயற்குழு பயன்பாட்டுக்கான நேரடி வலை UI ஐ இணைத்தல் ஆகியவற்றை செய்துள்ளீர்கள். இவை அனைத்தும் உங்கள் கருவியில் முழுமையாக இயங்குகின்றன.

---

## நீங்கள் எதை கட்டியுள்ளீர்கள்

| பகுதி | நீங்கள் கட்டியுள்ளீர்கள் |
|------|--------------------|
| 1 | Foundry Local ஐ நிறுவி, CLI மூலமாக மாதிரிகளை ஆராய்ந்தீர்கள் |
| 2 | Foundry Local SDK API ஐ சேவை, பட்டியல், கேஷ் மற்றும் மாதிரி மேலாண்மைக்கு கற்றுக்கொண்டீர்கள் |
| 3 | Python, JavaScript, மற்றும் C# மூலம் SDK பயன்படுத்தி OpenAI உடன் இணைந்தீர்கள் |
| 4 | உள்ளூர் அறிவுக் க்களவைக் கொண்ட RAG குழாய்முறை கட்டிய இடம் |
| 5 | தனித்துவமான AI முகவர்களை உருவாக்கி கட்டமைக்கப்பட்ட வெளியீட்டை உருவாக்கியது |
| 6 | பன்முக செயற்குழுக்களை கருத்துக்கணிப்பு வளையங்களுடன் ஒருங்கிணைத்தது |
| 7 | தயாரிப்பு முடிவுப் பயன்பாடான Zava Creative Writer ஐ ஆராய்ந்தது |
| 8 | முகவர்களுக்கு மதிப்பீடு வழிநடத்தும் மேம்பாட்டு பணிப்பட்டியலை கட்டியது |
| 9 | Whisper ஐ பயன்படுத்தி ஒலியை உரையாக்கியது, கருவியில் பேச்சு-உரை மாற்றம் |
| 10 | ONNX Runtime GenAI உடன் தனிப்பயன் Hugging Face மாதிரியை தொகுத்து இயக்கியது |
| 11 | கருவி அழைப்புடன் வெளிப்புற செயல்பாடுகளை அழைக்கும் உள்ளூர் மாதிரிகளை இயக்கு |
| 12 | நேரடி ஸ்ட்ரீமிங்குடன் Zava Creative Writer க்கான உலாவி அடிப்படையிலான UI ஐ கட்டியது |

## மேலதிக யோசனைகள்

- கருவி அழைப்பை முகவர்களுடன் இணைத்து சுயநிலை பணிகள் கட்டமைக்கவும்
- உங்கள் கருவிகளிலிருந்து உள்ளூர் தரவுத்தளத்தை விசாரிக்கவும் அல்லது உள் REST APIs ஐ அழைக்கவும்
- தரம் மற்றும் வேகத்தை ஒப்பிட `phi-4-mini`, `deepseek-r1-7b` போன்ற மாடல்களை முயற்சிக்கவும்
- உங்கள் Zava Writer வலைச் சேவையகத்திற்கு அங்கீகாரம் அல்லது வீதக் கட்டுப்பாடு சேர்க்கவும்
- நீங்கள் கவலைப்படுகிற பகுதியிற்கு உங்கள் சொந்த பன்முக செயற்குழு பயன்பாட்டை உருவாக்கவும்
- Foundry Local ஐ Azure AI Foundry க்கு மாற்றி மேகம் மூலம் இயக்கவும், அதே குறியீடு, வேறு முனை
