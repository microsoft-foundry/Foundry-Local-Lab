# အပိုင်း ၁၃: အလုပ်ရုံပြီးစီးခြင်း

> **ဂုဏ်ပြုပါတယ်!** သင်သည် Foundry Local အလုပ်ရုံအစုံကို ပြီးစီးလိုက်ပါပြီ။

သင်သည် CLI ထည့်သွင်းခြင်းမှစ၍ စကားပြောအက်ပ်များ၊ RAG ပိုင်းလိုင်းများ၊ မျိုးဆက် agents စနစ်များ၊ အသံမှ စာသားပြောင်းဖတ်ခြင်း၊ မိမိကိုယ်ပိုင်စိတ်တိုင်းကျမော်ဒယ်များ စုဆောင်းထုတ်လုပ်ခြင်း၊ ကိရိယာခေါ်ဆိုခြင်း ခွင့်ပြုခြင်း နှင့် မျိုးဆက် agents အက်ပ်အတွက် ရိုက် live အင်တာနက် UI ဖွဲ့စည်းခြင်းတို့ကို လုပ်ဆောင်နိုင်ခဲ့ပါသည်။ အားလုံးကို သင့်စက်ပေါ်တွင် ဆင်းစစ်အသုံးပြုထားသည်။

---

## သင်ဖန်တီးခဲ့သည့် အရာများ

| အပိုင်း | သင်ဖန်တီးခဲ့သည့်အရာ |
|------|---------------|
| 1 | Foundry Local ကို 설치 설치 설치 လုပ်ပြီး CLI မှတဆင့် မော်ဒယ်များကို စူးစမ်းမိန်းညို့လိုက်သည် |
| 2 | Foundry Local SDK API ကို ဝန်ဆောင်မှု၊ စာရင်းဇယား၊ cache နှင့် မော်ဒယ်စီမံခန့်ခွဲမှုအတွက် ကျွမ်းကျင်ခဲ့သည် |
| 3 | Python, JavaScript, နှင့် C# မှ SDK ဖြင့် OpenAI နှင့် ချိတ်ဆက်ခဲ့သည် |
| 4 | ဒေသတွင်း အသိပညာရှာဖွေရေးဖြင့် RAG ပိုင်းလိုင်း တည်ဆောက်ခဲ့သည် |
| 5 | ကိုယ်ပိုင် persona များနှင့် ဖွဲ့စည်းထားသော အထွက်ဖြင့် AI သွင်းးရုပ်များဖန်တီးခဲ့သည် |
| 6 | မျိုးဆက် agents ပိုင်းလိုင်းများကို ပြန်လည်တုံ့ပြန်မှု လုပ်ငန်းစဉ်များဖြင့် စီမံ ချုပ်ကိုက်ခဲ့သည် |
| 7 | ထုတ်လွှင့်ရေး ထုတ်ကုန် capstone အက်ပ်ဖြစ်သော Zava Creative Writer ကို စူးစမ်းလေ့လာခဲ့သည် |
| 8 | ဝန်ဆောင်မှုအတွက် အကဲဖြတ် ထောက်ပံ့ရေး ဖွံ့ဖြိုးတိုးတက်မှု လုပ်ငန်းစဉ်များတည်ဆောက်ခဲ့သည် |
| 9 | Whisper ဖြင့် အသံမှ စာသားပြောင်း ဆိုင်းခြင်း၊ စက်ပေါ်တွင် စကားပြောမှ စာသားပြောင်းခြင်း ပြုလုပ်ခဲ့သည် |
| 10 | ONNX Runtime GenAI နှင့် Hugging Face ကို ကိုယ်ပိုင်စုဆောင်းသည့် မော်ဒယ်ကို ချိတ်ဆွဲပြေးစီးခဲ့သည် |
| 11 | ကိရိယာခေါ်ဆိုခြင်း ဖြင့် ဒေသတွင်း မော်ဒယ်များကို ပြင်ပလုပ်ဆောင်ချက်များ ခေါ်ယူခွင့် ပြုခဲ့သည် |
| 12 | Zava Creative Writer အတွက် ပရောက်ဆာတွင် စက်တင်မရှိသော ဝဘ် UI ကို အချိန်နှင့်တပြေးညီ လှုပ်ရှားစေခဲ့သည် |

---

## နောက်ထပ် အတွေးအခေါ်များ

- ကိရိယာခေါ်ဆိုမှုနှင့် agents များကို ပူးပေါင်း၍ ကိုယ်ပိုင် လွတ်လပ်သော လုပ်ငန်းလည်ပတ်မှုများ ဆောက်လုပ်ပါ
- သင့်ကိရိယာများမှ ဒေသတွင်း ဒေတာဘေ့စ်မေးလ်ခြင်း သို့မဟုတ် အတွင်း REST API များကို ခေါ်ယူပါ
- အရည်အသွေး နှင့် အမြန်နှုန်းကို နှိုင်းယှဥ်ရန် မော်ဒယ် များကို စမ်းသပ်ပါ (`phi-4-mini`, `deepseek-r1-7b`)
- သင်၏ Zava Writer ဝဘ်ဆာဗာအတွက် အတိုးအကျသတ်မှတ်ခြင်း (authentication) သို့မဟုတ် အလျင်အမြန် ကန့်သတ်ခြင်း ထည့်သွင်းပါ
- သင်စိတ်ဝင်စားသော နယ်ပယ်အတွက် ကိုယ်ပိုင် မျိုးဆက် agents အက်ပ်တည်ဆောက်ပါ
- Foundry Local ကို Azure AI Foundry ဖြင့် နှစ်ဖက်စလုံး ပြောင်းလဲပြီး အဆိုပါ ကိုဒ်ကို အတူတူသုံးပြီး တီထွင်မှုများကို မတူညီသော endpoint မှ deploy လုပ်ပါ
