# Parte 13: Workshop Completato

> **Congratulazioni!** Hai completato l'intero Workshop Foundry Local.

Sei passato dall'installazione della CLI alla creazione di applicazioni di chat, pipeline RAG, sistemi multi-agente, trascrizione da voce a testo, compilazione dei tuoi modelli personalizzati, attivazione della chiamata di strumenti e creazione di un'interfaccia web live per la tua applicazione multi-agente. Tutto ciò in esecuzione completamente sul tuo dispositivo.
