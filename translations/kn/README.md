# Foundry Local ಕಾರ್ಯಾಗಾರ - ಸಾಧನದ ಮೇಲೆ AI ಅಪ್ಲಿಕೇಶನ್‌ಗಳನ್ನು ನಿರ್ಮಿಸಿ

ನಿಮ್ಮ ಸ್ವಂತ ಯಂತ್ರದಲ್ಲಿ ಭಾಷಾ ಮಾದರಿಗಳನ್ನು ಚಲಾಯಿಸಲು ಮತ್ತು [Foundry Local](https://foundrylocal.ai) ಮತ್ತು [Microsoft Agent Framework](https://learn.microsoft.com/en-us/agent-framework/) ಬಳಸಿ ಬುದ್ಧಿವಂತ ಅಪ್ಲಿಕೇಶನ್‌ಗಳನ್ನು ನಿರ್ಮಿಸುವ ಕೈಗೊಳ್ಳುವ ಕಾರ್ಯಾಗಾರ.

> **Foundry Local ಏನು?** Foundry Localವು ಒಂದು ಸಣ್ಣ ಒಟ್ಟಿಗೆ ಕಾರ್ಯಗತಗೊಳಿಸುವ_runtime_ ಆಗಿದ್ದು, ನಿಮ್ಮ ಹಾರ್ಡ್‌ವೇರ್‌ನಲ್ಲಿ ಸಂಪೂರ್ಣವಾಗಿ ಭಾಷಾ ಮಾದರಿಗಳನ್ನು ಡೌನ್‌ಲೋಡ್, ನಿರ್ವಹಣೆ ಮತ್ತು ಸೇವೆ ಮಾಡಲು ಅನುಮತಿಸುತ್ತದೆ. ಇದು **OpenAI-ಹೊಂದಿಕೊಳ್ಳಬಹುದಾದ API** ಅನ್ನು ಬಿಚ್ಚಿಡುತ್ತದೆ, ಹೀಗೆ ಯಾವುದೇ ಉಪಕರಣ ಅಥವಾ SDKವು OpenAI ಮಾತಾಡಿದರೆ ಸಂಪರ್ಕಿಸಬಹುದು - ಯಾವುದೇ ಕ್ಲೌಡ್ ಖಾತೆ ಅಗತ್ಯವಿಲ್ಲ.

## ಪ್ರಾರಂಭಿಸಲು

```bash
# 1. ರೆಪೊಸಿಟರಿಯನ್ನು ಕ್ಲೋನ್ ಮಾಡಿ
git clone https://github.com/microsoft-foundry/foundry-local-lab.git
cd foundry-local-lab

# 2. ಫೌಂಡ್ರಿ ಲೋಕಲ್‌ ಸ್ಥಾಪಿಸಲಾಗಿದೆ ಎಂದು ಪರಿಶೀಲಿಸಿ
foundry model list              # ಲಭ್ಯವಿರುವ ಮಾದರಿಗಳನ್ನು ಪಟ್ಟಿ ಮಾಡಿ
foundry model run phi-3.5-mini  # ಪ್ರಾಯೋಗಿಕ ಚಾಟ್ ಪ್ರಾರಂಭಿಸಿ

# 3. ನಿಮ್ಮ ಭಾಷಾ ಬಂಡಲವನ್ನು ಆಯ್ಕೆಮಾಡಿ (ಪೂರ್ಣ ಸೆಟಪ್‌ಗಾಗಿ ಭಾಗ 2 ಲ್ಯಾಬ್ ನೋಡಿ)
```

| ಭಾಷೆ | ದ್ರುತ ಪ್ರಾರಂಭ |
|--------|---------------|
| **Python** | `cd python && pip install -r requirements.txt && python foundry-local.py` |
| **C#** | `cd csharp && dotnet run` |
| **JavaScript** | `cd javascript && npm install && node foundry-local.mjs` |

## ಕಾರ್ಯಾಗಾರ ಭಾಗಗಳು

### ಭಾಗ 1: Foundry Local ನೊಂದಿಗೆ ಪ್ರಾರಂಭಿಸುವುದು

** ಪ್ರಯೋಗಮಾನ ಸಲಹೆ:** [`labs/part1-getting-started.md`](labs/part1-getting-started.md)

- Foundry Local ಏನು ಮತ್ತು ಅದು ಹೇಗೆ ಕೆಲಸ ಮಾಡುತ್ತದೆ
- Windows ಮತ್ತು macOS ನಲ್ಲಿ CLI ಅನ್ನು ಸ್ಥಾಪಿಸುವುದು
- ಮಾದರಿಗಳನ್ನು ಅನ್ವೇಷಿಸುವುದು - ಪಟ್ಟಿ ಮಾಡುವಿಕೆ, ಡೌನ್‌ಲೋಡ್, ಚಾಲನೆ
- ಮಾದರಿ ಉಪನಾಮಗಳು ಮತ್ತು ಡೈನಾಮಿಕ್ ಪೋರ್ಟುಗಳನ್ನು ಅರ್ಥಮಾಡಿಕೊಳ್ಳುವುದು

---

### ಭಾಗ 2: Foundry Local SDK ಅನ್ನು ಆಳವಾಗಿ ಅರಿಯುವುದು

** ಪ್ರಯೋಗಮಾನ ಸಲಹೆ:** [`labs/part2-foundry-local-sdk.md`](labs/part2-foundry-local-sdk.md)

- ಅಪ್ಲಿಕೇಶನ್ ಅಭಿವೃದ್ಧಿಗಾಗಿ CLI ಬದಲು SDK ಯನ್ನು ಯಾಕೆ ಬಳಸಬೇಕು
- Python, JavaScript, ಮತ್ತು C# ಗೆ SDK API ಸಂಪೂರ್ಣ ರೆಫರೆನ್ಸ್
- ಸೇವೆ ನಿರ್ವಹಣೆ, ಕ್ಯಾಟಲಾಗ್ ಬ್ರೌಸಿಂಗ್, ಮಾದರಿ ಬದುಕುಚಕ್ರ (ಡೌನ್‌ಲೋಡ್, ಲೋಡ್, ಅನ್ಲೋಡ್)
- ದ್ರುತ ಪ್ರಾರಂಭ ಮಾದರಿಗಳು: Python ನಿರ್ಮಾಪಕ ಬೂಟ್ಸ್ಟ್ರಾಪ್, JavaScript `init()`, C# `CreateAsync()`
- `FoundryModelInfo` ಮೆಟಾಡೇಟಾ, ಉಪನಾಮಗಳು, ಮತ್ತು ಹಾರ್ಡ್‌ವೇರ್-ಉತ್ತಮ ಮಾದರಿ ಆಯ್ಕೆ
