# ಭಾಗ 7 - ವಿಪ್ಸ್ ಧ್ವನಿ ಲಿಪ್ಯಂತರಣಕ್ಕೆ ಮಾದರಿ ಧ್ವನಿ ಕಡತಗಳು

ಈ WAV ಕಡತಗಳು `pyttsx3` (ವಿಂಡೋಸ್ SAPI5 ಪಠ್ಯದಿಂದ ಧ್ವನಿಗೆ) ಬಳಸಿ ರಚಿಸಲಾಗಿದೆ ಮತ್ತು ಕ್ರಿಯೇಟಿವ್ ರೈಟರ್ ಡೆಮೋದಿಂದ **Zava DIY ಉತ್ಪನ್ನಗಳು** üzerine ಥೀಮ್ ಮಾಡಲಾಗಿದೆ.

## ಮಾದರಿಗಳನ್ನು ಉತ್ಪಾದಿಸಿ

```bash
# ರೆಪೋ ರೂಟ್‌ನಿಂದ - pyttsx3 ಅನ್ನು ಸ್ಥಾಪಿಸಿರುವ .venv ಅವಶ್ಯಕವಾಗಿದೆ
.venv\Scripts\Activate.ps1          # ವಿಂಡೋಸ್
python samples/audio/generate_samples.py
```

## σημειώσεις

- WAV ಕಡತಗಳು ರೆಪೊಗೆ **ಕಮಿಟ್** ಮಾಡಲ್ಪಟ್ಟಿವೆ (`. ನಲ್ಲಿ ಪಟ್ಟಿ ಮಾಡಲಾಗಿದೆ). ಹೊಸ .wav ಕಡತಗಳನ್ನು ರಚಿಸಲು ಮೇಲಿನ ಸ್ಕ್ರಿಪ್ಟ್ ನಡಿಸಿ ಅಥವಾ ಹೊಸ ಸ್ಕ್ರಿಪ್ಟ್‌ಗಳನ್ನು ತಯಾರಿಸಲು ಸಂಪಾದಿಸಿ.
- ಸ್ಕ್ರಿಪ್ಟ್ **Microsoft David** (US ಇಂಗ್ಲಿಷ್) ಧ್ವನಿಯನ್ನು 160 WPM ನಲ್ಲಿ ಸ್ಪಷ್ಟ ಲಿಪ್ಯಂತರಣ ಫಲಿತಾಂಶಗಳಿಗಾಗಿ ಬಳಸುತ್ತದೆ.
- ಎಲ್ಲಾ ಸ್ಥಳಕಥನಗಳು [`zava-creative-writer-local/src/api/agents/writer/products.json`](../../../../zava-creative-writer-local/src/api/agents/writer/products.json) ನಿಂದ ಉತ್ಪನ್ನಗಳ ಕುರಿತು ಉಲ್ಲೇಖಿಸುತ್ತವೆ.
